# 遊戲概述

## 1.1 一句話描述

> 以魔法師身份攀登百層高塔，透過收集魔法卷軸與培養使魔，以回合制戰鬥征服每一層的冒險RPG。

## 1.2 遊戲類型

- **主類型**：回合制 RPG
- **次類型**：Roguelite（程序生成樓層內容 + 永久進度保留）
- **靈感來源**：
  - 寶可夢 1~9 代（回合制技能戰鬥、屬性相剋、大量技能池）
  - Sword Art Online S1（100層高塔、安全區城鎮、層層攻略）
  - Slay the Spire（節點式地圖選擇、build構築）
  - 不思議迷宮（點擊式探索、事件選項）

## 1.3 目標玩家

- 喜歡回合制策略戰鬥的玩家
- 喜歡收集、培養、構築 build 的玩家
- 喜歡逐步攻略、有明確進度目標的玩家
- 不擅長或不喜歡動作操作的玩家

## 1.4 核心賣點

1. **龐大的技能池**（200+ 魔法技能）構築無限 build 組合
2. **100 層高塔**提供明確的長期目標感
3. **使魔培養**增加戰鬥夥伴養成樂趣
4. **純點擊操作**，所有互動透過選項完成，零操作門檻
5. **幾何極簡美術風格**，用 UI 品質取代美術素材

## 1.5 操作方式

- **僅使用滑鼠點擊**
- 所有探索、戰鬥、城鎮互動皆透過畫面上的按鈕/選項完成
- 支援鍵盤快捷鍵（數字鍵 1~6 對應技能格）作為輔助

## 2. 核心設計理念

### 2.1 決策驅動

> 遊戲的每一刻都在做有意義的選擇

- 探索時：選擇走哪條路線、如何處理事件
- 戰鬥時：選擇使用哪個技能、何時用道具、何時逃跑
- 成長時：選擇學哪個技能、帶哪隻使魔、裝哪個飾品

### 2.2 Build 構築

> 每次攀塔都能嘗試不同的流派組合

- 技能格有限（6主動+2被動），逼玩家專精
- 飾品（3格）定義核心流派
- 使魔類型補強或互補
- 沒有「最優解」，鼓勵實驗

### 2.3 風險與報酬

> 越危險的選擇，越豐厚的回報

- 精英戰難但掉落稀有
- 事件中大膽的選項可能大賺或大虧
- PP/MP 資源管理：強技能消耗大，何時用是關鍵
- 繼續推進 vs 回安全區補給的抉擇

### 2.4 漸進式複雜度

> 前 10 層教會基礎，後 90 層逐步加深

- 1~9F：只有基礎戰鬥和簡單事件
- 10~30F：引入飾品、使魔、屬性相剋
- 30~60F：引入技能突變、精英戰、複合屬性敵人
- 60~100F：全系統開放，高難度挑戰
