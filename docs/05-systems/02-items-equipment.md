## 11.3 裝備系統

### 11.3.1 法杖（武器）

法杖影響**魔法攻擊力**和**元素加成**。

**法杖屬性：**
- M.ATK（魔法攻擊力加成）
- 元素加成（特定元素傷害+X%）
- 特殊效果（部分高級法杖）

**法杖品質等級：**

| 品質 | 顏色 | M.ATK範圍 | 出現樓層 |
|------|------|----------|---------|
| 普通 | 白色 | +5~15 | 1F~20F |
| 優良 | 綠色 | +15~30 | 10F~40F |
| 稀有 | 藍色 | +30~50 | 30F~60F |
| 史詩 | 紫色 | +50~80 | 50F~80F |
| 傳說 | 金色 | +80~120 | 70F~100F |

**法杖範例：**

| 名稱 | 品質 | M.ATK | 特殊效果 |
|------|------|-------|---------|
| 見習法杖 | 普通 | +8 | 無 |
| 火焰法杖 | 優良 | +20 | 火系+15% |
| 暴風之杖 | 稀有 | +38 | 風系+20%，速度+10 |
| 深淵權杖 | 史詩 | +65 | 暗系+30%，吸血5% |
| 亞克圖斯之杖 | 傳說 | +100 | 全元素+15%，MP消耗-10% |

### 11.3.2 法袍（防具）

法袍影響**防禦力**和**元素抗性**。

**法袍屬性：**
- M.DEF（魔法防禦力加成）
- P.DEF（物理防禦力加成）
- 元素抗性（特定元素傷害減免X%）
- HP/MP 加成（部分法袍）

**法袍範例：**

| 名稱 | 品質 | M.DEF | P.DEF | 特殊效果 |
|------|------|-------|-------|---------|
| 見習法袍 | 普通 | +5 | +3 | 無 |
| 火焰抗性袍 | 優良 | +15 | +8 | 火系抗性+20% |
| 元素外套 | 稀有 | +30 | +15 | 全元素抗性+10% |
| 大賢者之袍 | 史詩 | +55 | +25 | MP+100，魔力再生 |
| 星辰法袍 | 傳說 | +85 | +40 | 全抗性+20%，HP+200 |

### 11.3.3 強化系統

裝備可在鍛造工坊進行強化（+1 ~ +10）：

```
每次強化提升：
  法杖：M.ATK + 基礎值的 8%
  法袍：M.DEF + 基礎值的 8%，P.DEF + 基礎值的 8%

強化消耗：
  +1: 強化石x1 + 100G
  +2: 強化石x2 + 200G
  +3: 強化石x3 + 400G
  +4: 強化石x5 + 800G
  +5: 強化石x8 + 1500G
  +6: 高級強化石x1 + 3000G（80%成功率）
  +7: 高級強化石x2 + 5000G（80%成功率）
  +8: 高級強化石x3 + 8000G（60%成功率）
  +9: 高級強化石x5 + 12000G（60%成功率）
  +10: 傳說強化石x1 + 20000G（40%成功率）
```

## 11.4 素材系統

素材從地牢掉落或事件取得，用於鍛造、合成、培養。

**素材分類：**

| 類型 | 範例 | 用途 |
|------|------|------|
| 元素結晶 | 火之結晶、水之結晶... | 技能突變、元素裝備製造 |
| 金屬礦石 | 鐵礦、秘銀礦、星辰礦 | 裝備製造 |
| 強化材料 | 強化石、高級強化石 | 裝備強化 |
| 魔物素材 | 精靈核心、龍鱗、蛇皮 | 飾品合成、使魔培養 |
| 草藥素材 | 回復草、魔力花、稀有菇 | 藥水調製 |
| 特殊素材 | 古代碎片、封印之鑰 | 傳說裝備、隱藏內容 |
