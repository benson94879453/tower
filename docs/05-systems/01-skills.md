# 魔法技能系統

## 10.1 技能概要

技能是本遊戲的**核心收集要素**，目標提供 200+ 個技能供玩家收集與構築 build。

**技能格限制：**
- 主動技能格：**6 格**
- 被動技能格：**2 格**
- 技能背包（已學但未裝備）：**無上限**

## 10.2 技能獲取途徑

| 途徑 | 說明 | 取得難度 |
|------|------|---------|
| **魔法卷軸** | 地牢掉落/商店購買，使用後直接學會 | 低~中 |
| **魔法書** | Boss掉落/圖書館合成，可反覆研讀提升熟練度 | 中~高 |
| **被動領悟** | 被敵人技能擊中後 5% 機率領悟 | 低（隨機） |
| **圖書館研究** | 消耗研究點數+素材，合成新魔法 | 中 |
| **任務獎勵** | 完成特定任務獲得 | 固定 |
| **事件獎勵** | 探索中特定事件選項 | 隨機 |
| **使魔共鳴** | 使魔進化時可能傳授技能給主角 | 稀有 |

## 10.3 技能成長系統

每個技能有**熟練度等級**（Lv.1 ~ Lv.MAX）：

```
學會技能 -> Lv.1（基礎版）
  |
  使用技能/研讀魔法書 累積熟練度
  |
  v
Lv.2（威力/效果小幅提升）
  |
  v
Lv.3（威力/效果中幅提升，可能解鎖附加效果）
  |
  v
Lv.MAX（完全精通）
  |
  v
突變（分支進化二選一，不可逆）
```

**熟練度獲取：**
- 戰鬥中使用該技能：+1~3 熟練度（擊殺敵人+1額外）
- 研讀對應魔法書：+10 熟練度
- 圖書館研究加速：+5 熟練度（消耗研究點數）

**升級所需熟練度：**
- Lv.1 -> Lv.2：50 熟練度
- Lv.2 -> Lv.3：120 熟練度
- Lv.3 -> Lv.MAX：250 熟練度
- Lv.MAX -> 突變：500 熟練度 + 特殊素材

## 10.4 技能突變系統

達到 Lv.MAX 後，滿足條件可進行**突變**（技能進化）。每個技能有**兩條突變路線**，選擇後不可逆。

**範例：火球術的突變路線**

```
火球術 Lv.MAX
  |
  +--[突變A] 隕石術
  |   威力: 65 -> 120
  |   範圍: 單體 -> 全體
  |   MP: 12 -> 35
  |   附加: 30%機率燃燒
  |   冷卻: 2回合
  |
  +--[突變B] 爆裂火球
      威力: 65 -> 150
      範圍: 單體
      MP: 12 -> 28
      附加: 必定燃燒 + 防禦下降
      冷卻: 無
```

**突變條件：**
- 基礎條件：熟練度達到 500
- 特殊素材：如「火之結晶 x5」（隕石術）或「爆裂核心 x3」（爆裂火球）
- 部分突變需要額外條件：如「累計用火球術擊敗 100 隻敵人」

## 10.5 技能分類

### 按功能分類

| 分類 | 說明 | 技能數量目標 |
|------|------|------------|
| 單體攻擊 | 對單一目標造成傷害 | 每元素 5~8 個 |
| 全體攻擊 | 對所有敵人造成傷害 | 每元素 2~3 個 |
| 持續傷害 | 施加DoT（燃燒/毒等） | 每元素 1~2 個 |
| 狀態賦予 | 施加控制/弱化狀態 | 15~20 個 |
| 自身增益 | 提升自身能力 | 10~15 個 |
| 敵方減益 | 降低敵方能力 | 10~15 個 |
| 回復 | 恢復HP/MP/狀態 | 8~12 個 |
| 防禦 | 護盾/反射/減傷 | 8~10 個 |
| 特殊 | 獨特機制的技能 | 15~20 個 |

## 10.6 被動技能

被動技能不需要手動使用，裝備後永久生效。

**被動技能範例：**

| 技能名 | 效果 | 取得 |
|--------|------|------|
| 火焰親和 | 火系技能傷害+15% | 熔岩區域掉落 |
| 冰霜護體 | 受到物理攻擊時10%機率凍結攻擊者 | 冰域Boss掉落 |
| 魔力湧泉 | 每回合回復 2% MP | 圖書館研究 |
| 快速詠唱 | 所有技能冷卻 -1 回合 | 50F+ 稀有掉落 |
| 生命汲取 | 造成傷害的 8% 轉為HP回復 | 暗影區域掉落 |
| 元素大師 | 屬性相剋效果絕佳時傷害額外 +25% | 通關後隱藏任務 |
| 使魔共鳴 | 使魔攻擊力+20% | 使魔小屋特殊事件 |
| 幸運星 | 稀有掉落率+10% | 競技場兌換 |

## 10.7 冷卻系統

部分強力技能有冷卻時間（CD）：

- 使用後需等待 X 回合才能再次使用
- 冷卻期間技能格顯示為灰色+剩餘回合數
- 被動技能「快速詠唱」可減少冷卻
- 部分飾品可減少特定元素技能冷卻

## 10.8 PP 系統（技能使用次數）

每個技能有 PP（Power Points）上限，代表單次探索中的使用次數：

- 低威力技能：PP 15~20
- 中威力技能：PP 8~12
- 高威力技能：PP 3~5
- 回復 PP 方式：
  - 回到安全區（全回復）
  - 使用「以太」道具（回復單個技能 PP）
  - 中繼點研讀魔法書（回復該書對應技能 PP）
