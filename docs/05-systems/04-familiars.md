# 使魔系統

## 13.1 使魔概要

使魔是主角的戰鬥夥伴，在戰鬥中自動行動（由 AI 控制）。玩家可透過指令切換使魔的行為模式。

**基礎規則：**
- 攜帶上限：**1 隻**（進入地牢前選擇）
- 存放上限：**10 隻**（使魔小屋）
- 使魔有獨立的 HP、屬性、技能
- 使魔 HP 歸零 -> 退場（本次探索不可再出場，回安全區自動回復）

## 13.2 使魔獲取

**獲取方式：**
1. 擊敗特定敵人 -> 掉落「使魔核心」
2. 攜帶使魔核心回到安全區「使魔小屋」
3. 消耗素材 + 金幣進行「孵化」
4. 孵化後獲得對應使魔

**使魔核心掉落率：**
- 普通敵人：1%
- 精英敵人：5%
- Boss：15%（首次擊敗 100%）

## 13.3 使魔類型

| 類型 | 特點 | 幾何外觀 | 代表使魔 |
|------|------|---------|---------|
| **攻擊型** | 高攻擊，低防禦 | 紅色尖銳三角形/菱形 | 火焰小鬼、雷光精靈 |
| **防禦型** | 高防禦，可替主角擋傷 | 藍色方形/六角形 | 岩石守衛、冰晶護衛 |
| **輔助型** | 每回合提供回復/增益 | 綠色圓形/橢圓 | 森林妖精、光之精靈 |
| **速攻型** | 極高速度，連擊能力 | 黃色星形/閃電形 | 風之使者、影子刺客 |
| **特殊型** | 獨特能力（偵測/掉落等） | 紫色星形/不規則形 | 虛空觀測者、幸運貓 |

## 13.4 使魔屬性

每隻使魔有以下屬性：

```
名稱：火焰小鬼
類型：攻擊型
元素：火
等級：Lv.1（最高 Lv.50）

HP:  80        M.ATK: 35
MP:  30        M.DEF: 12
P.ATK: 20     P.DEF: 10
速度: 28

技能格：2 個
技能1：火焰爪（火，威力40，PP15）
技能2：（空）

進化條件：Lv.25 + 火之結晶x10
進化後：獄炎魔
```

## 13.5 使魔成長

### 等級提升
- 使魔跟隨主角戰鬥獲得經驗值（主角獲得量的 50%）
- 升級提升基礎屬性
- 每 10 級解鎖一個里程碑獎勵

### 餵養培養
- 在使魔小屋消耗素材餵養使魔
- 不同素材提升不同屬性：
  - 力量果實 -> M.ATK +2
  - 守護果實 -> M.DEF +2, P.DEF +2
  - 速度果實 -> 速度 +2
  - 生命果實 -> HP +10
  - 元素結晶 -> 對應元素傷害 +3%
- 每隻使魔有餵養上限（防止無限堆疊）：每種屬性最多 +20

### 技能教學
- 使用魔法卷軸可以教使魔新技能
- 使魔技能格只有 **2 個**，需要覆蓋舊技能
- 使魔只能學習與自身元素相同或無屬性的技能

## 13.6 使魔進化

滿足條件後可在使魔小屋進行進化：

```
進化條件 = 等級要求 + 特殊素材

範例進化鏈：
火焰小鬼 (Lv.1) 
  -> Lv.25 + 火之結晶x10 
  -> 獄炎魔 (進化)
     -> Lv.45 + 熔岩核心x5 + 龍鱗x3
     -> 獄炎魔王 (最終進化)
```

**進化效果：**
- 大幅提升基礎屬性
- 外觀變化（更大/更複雜的幾何形狀）
- 可能習得新技能
- 可能改變/增加元素屬性
- 部分進化路線有分支（二選一）

## 13.7 使魔戰鬥 AI

使魔根據玩家設定的指令自動行動：

**攻擊模式：**
```
1. 優先使用屬性相剋有利的攻擊技能
2. 如果沒有有利屬性，使用威力最高的攻擊技能
3. PP 不足時使用普通攻擊
```

**防禦模式：**
```
1. 本回合替主角承受下一次攻擊（傷害由使魔承受）
2. 如果使魔 HP < 30%，自動切換為待命模式
```

**輔助模式：**
```
1. 主角 HP < 50% 時優先使用回復技能
2. 否則使用增益技能
3. 沒有輔助技能時使用普通攻擊
```

**待命模式：**
```
不行動，節省 PP
```
