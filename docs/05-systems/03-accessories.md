# 飾品系統

## 12.1 飾品概要

飾品是定義玩家 **build 流派**的核心裝備。每個飾品提供**獨特的被動效果**，3 個飾品欄的組合決定了戰鬥風格。

## 12.2 飾品欄位

- 裝備欄：**3 個飾品格**
- 飾品不可強化，但可以在鍛造工坊合成
- 同名飾品不可重複裝備

## 12.4 飾品 Build 組合範例

### 「核爆火法」Build
- 燃燒之戒 + 過載符文 + 暴擊護符
- 配合高威力火系技能，追求單回合最大傷害
- 優點：爆發極高
- 缺點：MP消耗巨大，續航差

### 「永動機」Build
- 生命虹吸 + 魔力之泉 + 節能寶珠
- 配合中等威力技能持續戰鬥
- 優點：幾乎不需要補給
- 缺點：單回合輸出較低

### 「使魔大師」Build
- 使魔共鳴石 + 雙重行動 + 靈魂鏈接
- 主角以輔助為主，使魔為主要輸出
- 優點：使魔傷害極高，主角安全
- 缺點：使魔被擊敗則戰力大減

### 「毒霧控制」Build
- 詛咒面具 + 中毒增幅 + 連鎖詛咒
- 以狀態異常為主要傷害來源
- 優點：對高防敵人有效
- 缺點：對免疫狀態的Boss無效
