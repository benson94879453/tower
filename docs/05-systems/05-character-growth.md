# 角色成長系統

## 16.1 主角屬性

主角「艾倫」的基礎屬性：

```
等級：Lv.1 ~ Lv.99
HP:  100（基礎）     每級 +12
MP:  60（基礎）      每級 +8
M.ATK: 15（基礎）    每級 +3
M.DEF: 10（基礎）    每級 +2
P.ATK: 8（基礎）     每級 +1
P.DEF: 8（基礎）     每級 +1.5
速度: 15（基礎）     每級 +1.5
命中: 100（基礎）    每級 +0.5
迴避: 5（基礎）      每級 +0.3
暴擊率: 5%（基礎）   不隨等級成長（靠裝備/飾品提升）
```

## 16.2 經驗值與升級

**升級所需經驗值公式：**
```
所需EXP(Lv) = 50 + (Lv^2) x 5
```

| 等級 | 所需EXP | 累計EXP |
|------|---------|---------|
| 1→2 | 55 | 55 |
| 5→6 | 175 | 675 |
| 10→11 | 550 | 3,300 |
| 20→21 | 2,050 | 14,350 |
| 30→31 | 4,550 | 45,650 |
| 50→51 | 12,550 | 214,550 |
| 70→71 | 24,550 | 589,550 |
| 99→MAX | — | ~1,650,000 |

**經驗值來源：**
- 戰鬥勝利（主要來源）
- 任務完成獎勵
- 事件選項獎勵
- 競技場勝利
