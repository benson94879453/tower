# NPC 與任務系統

## 17.1 任務類型

| 類型 | 發布方式 | 說明 |
|------|---------|------|
| **主線任務** | 自動觸發 | 推進劇情，通關各Boss |
| **支線任務** | NPC對話觸發 | 探索特定內容，獲得獎勵 |
| **懸賞任務** | 酒館任務板 | 可重複，擊敗特定敵人/收集素材 |
| **隱藏任務** | 特殊條件觸發 | 高獎勵，需要特定條件組合 |

## 17.2 主線任務流程

```
第一章：覺醒（1F~9F）
  - 學習基本操作
  - 擊敗石像守衛
  - 獲得第一隻使魔

第二章：火之試煉（10F~19F）
  - 穿越熔岩鍛爐
  - 遇到導師留下的線索
  - 擊敗熔岩巨蛇

第三章：冰與雷的洗禮（20F~39F）
  - 發現塔的歷史碎片
  - 遇到其他挑戰者的殘跡
  - 連續擊敗冰霜女巫、風暴飛龍

第四章：大地的根基（40F~49F）
  - 發現導師的研究筆記
  - 了解塔之核心的秘密
  - 擊敗大地巨像

第五章：雷霆之上（50F~59F）
  - 競技場解鎖，遇到強大的對手
  - 發現塔的真正目的
  - 擊敗雷神雙子

第六章：光與暗的抉擇（60F~79F）
  - 劇情分歧點
  - 選擇是否相信導師的理念
  - 擊敗暗影領主、聖騎士長

第七章：虛空的真相（80F~89F）
  - 發現導師亞克圖斯的真正意圖
  - 決定最終立場
  - 擊敗虛空吞噬者

第八章：登頂（90F~100F）
  - 最終決戰準備
  - 面對亞克圖斯的分身
  - 挑戰塔之核心
  - 結局選擇
```

## 17.4 懸賞任務系統

酒館任務板每次休息後刷新 3 個懸賞任務：

**懸賞類型：**
- **討伐型**：擊敗指定敵人 x 數量
- **收集型**：收集指定素材 x 數量
- **探索型**：到達指定樓層/發現特定事件

**獎勵公式：**
```
金幣 = 基礎金幣 x 樓層修正 x 難度修正
EXP = 基礎EXP x 樓層修正 x 難度修正
額外獎勵 = 稀有任務有機率給特殊素材/卷軸
```

## 17.5 隱藏任務

隱藏任務不會在任務列表中顯示，需要玩家自己發現觸發條件。

| 隱藏任務 | 觸發條件 | 獎勵 |
|---------|---------|------|
| 幸運的邂逅 | 在事件中救助迷路冒險者 3 次 | 幸運貓使魔核心 |
| 元素之子 | 同時裝備全7元素技能 | 傳說被動「元素大師」 |
| 前代勇者之路 | 收集全部10本導師筆記 | 傳說法杖「初代勇者之杖」 |
| 黑市入場券 | 被神秘商人拒絕 5 次後 | 黑市通行證飾品 |
| 真正的守護者 | 使魔替主角擋下致命攻擊 10 次 | 靈魂鏈接飾品 |
| 賭徒之路 | 在祭壇獻祭 20 次 | 賭徒之幣飾品 |
