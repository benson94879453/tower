## 21.3 幾何圖形設計規範

### 角色/敵人表示

所有角色和敵人使用 **SVG 風格幾何圖形**：

```
主角：
  - 基礎形狀：正五邊形
  - 內部：星形紋路（代表魔法師）
  - 顏色：根據當前裝備元素主色
  - 大小：中型（48x48 基準）

普通敵人：
  - 基礎形狀：根據類型（三角/方/圓/菱等）
  - 內部：簡單紋路
  - 顏色：元素主色
  - 大小：小型~中型（32x32 ~ 48x48）

精英敵人：
  - 基礎形狀：複雜幾何（六角/星等）
  - 內部：發光紋路
  - 顏色：元素主色 + 金色邊框
  - 大小：中型~大型（48x48 ~ 64x64）

Boss：
  - 基礎形狀：獨特複合幾何
  - 內部：動態紋路/動畫
  - 顏色：元素色 + 特效光暈
  - 大小：大型（96x96 ~ 128x128）

使魔：
  - 基礎形狀：圓潤幾何（親和感）
  - 內部：類型標記（攻/防/輔）
  - 顏色：元素色（較柔和）
  - 大小：小型（32x32）
```

### 技能特效

技能特效使用 **粒子系統 + 幾何圖形動畫**：

| 元素 | 粒子形狀 | 動畫效果 |
|------|---------|---------|
| 火 | 三角形粒子 | 向上飄散，尺寸遞減 |
| 水 | 圓形/水滴粒子 | 波浪擴散 |
| 雷 | 閃電線段 | 快速閃爍+分支 |
| 風 | 弧線粒子 | 螺旋旋轉 |
| 土 | 方形碎片 | 從下方彈起 |
| 光 | 星形/十字粒子 | 向外爆發 |
| 暗 | 不規則形粒子 | 向內收縮 |
