# 回合制戰鬥系統

## 8.1 戰鬥概要

採用經典的**寶可夢式 1v1 / 1v多 回合制戰鬥**，玩家方（主角+使魔）vs 敵人方（1~3隻敵人）。

## 8.2 回合流程

```
回合開始
  |
  v
[速度排序] -> 決定所有行動者的出手順序
  |
  v
[玩家選擇行動]
  - 使用技能（選擇目標）
  - 使用道具
  - 切換使魔指令（攻擊/防禦/輔助）
  - 逃跑
  |
  v
[使魔自動行動] -> 根據指令AI自動選擇技能
  |
  v
[敵人行動] -> 根據AI選擇技能
  |
  v
[結算階段]
  - 傷害計算
  - 狀態異常判定（施加/解除）
  - 持續傷害/回復結算（燃燒、中毒、再生等）
  - 環境效果結算
  - buff/debuff 回合數 -1
  |
  v
[勝負判定]
  - 所有敵人 HP <= 0 -> 勝利
  - 玩家 HP <= 0 -> 敗北
  - 否則 -> 下一回合
```

## 8.3 速度系統

- 每個角色有「速度」屬性
- 每回合按速度由高到低排序行動
- **速度相同**時隨機決定先後
- **優先度**：某些技能有優先度修正
  - +1 優先度：快速技能（如「快速治癒」「先制攻擊」）
  - +2 優先度：防禦/保護技能
  - -1 優先度：蓄力技能
  - 優先度高的技能無視速度，必定先出手

## 8.4 戰鬥行動詳細

### 8.4.1 使用技能

- 從已裝備的 **6 個主動技能** 中選擇一個
- 消耗 **MP**（魔力值）和 **PP**（技能使用次數）
- 選擇目標（單體技能需選目標，全體技能自動對全體）
- 技能效果立即計算或延遲（蓄力技能）

### 8.4.2 使用道具

- 從背包中選擇消耗品使用
- 使用道具**不消耗MP**但**佔用一個回合**
- 常用戰鬥道具：
  - 回復藥水（回復HP）
  - 魔力藥水（回復MP）
  - 解毒草（解除中毒）
  - 萬靈藥（解除所有狀態異常）
  - 力量藥劑（攻擊力+30%，3回合）
  - 防禦藥劑（防禦力+30%，3回合）
  - 煙霧彈（逃跑成功率100%）

### 8.4.3 使魔指令

玩家可以切換使魔的行動模式（切換不消耗回合，但每回合只能切換一次）：

| 指令 | 使魔行為 |
|------|---------|
| **攻擊** | 優先使用攻擊技能 |
| **防禦** | 替主角承受下一次攻擊（一次性） |
| **輔助** | 優先使用回復/增益技能 |
| **待命** | 不行動（保存PP） |

### 8.4.4 逃跑

- 基礎逃跑成功率：**40%**
- 速度高於敵人：+20%
- 速度低於敵人：-10%
- 使用煙霧彈：100%
- Boss戰：**不可逃跑**
- 逃跑失敗：本回合無行動，敵人正常行動

## 8.5 傷害計算公式

```
基礎傷害 = (攻擊力 x 技能威力 / 100) - (對方防禦力 / 2)
屬性修正 = 基礎傷害 x 屬性倍率（見第9章）
暴擊修正 = 屬性修正 x 暴擊倍率（暴擊時 x1.5，否則 x1.0）
隨機修正 = 暴擊修正 x random(0.85, 1.00)
最終傷害 = max(1, 隨機修正)  // 最少造成1點傷害
```

**攻擊力區分：**
- **魔法攻擊力（M.ATK）**：影響魔法技能傷害
- **物理攻擊力（P.ATK）**：影響物理技能傷害（少數技能）
- **魔法防禦力（M.DEF）**：減免魔法傷害
- **物理防禦力（P.DEF）**：減免物理傷害

## 8.6 命中與迴避

```
命中率 = 技能命中率 x (自身命中修正 / 對方迴避修正)
最終命中 = clamp(命中率, 5%, 100%)  // 最低5%命中，最高100%
```

- 部分技能為「必中」（無視迴避）
- 部分技能為「一擊必殺」（命中率固定30%，命中則直接擊敗）

## 8.7 暴擊系統

```
暴擊率 = 基礎暴擊率(5%) + 裝備加成 + 技能加成 + buff加成
暴擊倍率 = 1.5（基礎）+ 飾品加成
```

- 部分技能有「高暴擊率」特性（暴擊率+25%）
- 飾品可以提升暴擊率或暴擊傷害

## 8.8 狀態異常系統

### 持續傷害型

| 狀態 | 效果 | 持續 | 施加源 |
|------|------|------|--------|
| 燃燒 | 每回合損失 8% 最大HP | 3回合 | 火系技能 |
| 中毒 | 每回合損失 6% 最大HP | 5回合 | 毒系技能/陷阱 |
| 猛毒 | 每回合損失 12% 最大HP | 3回合 | 高階毒技能 |

### 控制型

| 狀態 | 效果 | 持續 | 解除 |
|------|------|------|------|
| 凍結 | 無法行動 | 1~2回合 | 被火系攻擊時解除 |
| 麻痺 | 50%機率無法行動 | 3回合 | 自然解除或道具 |
| 睡眠 | 無法行動 | 2~4回合 | 受到攻擊時解除 |
| 混亂 | 50%機率攻擊自己 | 2~3回合 | 自然解除或道具 |
| 魅惑 | 50%機率攻擊隊友 | 2回合 | 自然解除或道具 |

### 弱化型

| 狀態 | 效果 | 持續 |
|------|------|------|
| 攻擊下降 | 攻擊力-25% | 3回合 |
| 防禦下降 | 防禦力-25% | 3回合 |
| 速度下降 | 速度-25% | 3回合 |
| 命中下降 | 命中率-25% | 3回合 |
| 封印 | 隨機1個技能不可用 | 2回合 |
| 詛咒 | 回復效果反轉為傷害 | 3回合 |

### 增益型

| 狀態 | 效果 | 持續 |
|------|------|------|
| 攻擊提升 | 攻擊力+25% | 3回合 |
| 防禦提升 | 防禦力+25% | 3回合 |
| 速度提升 | 速度+25% | 3回合 |
| 再生 | 每回合回復 5% 最大HP | 5回合 |
| 魔力再生 | 每回合回復 3% 最大MP | 5回合 |
| 護盾 | 吸收一定量傷害 | 直到護盾值歸零 |
| 反射 | 反射下一次魔法攻擊 | 1次 |
| 隱身 | 下一次攻擊必閃 | 1次 |

## 8.9 戰鬥勝利獎勵

| 獎勵類型 | 計算方式 |
|----------|---------|
| 經驗值 | 敵人基礎EXP x 數量修正 x 樓層修正 |
| 金幣 | 敵人基礎Gold x random(0.8, 1.2) |
| 掉落物 | 每隻敵人獨立判定掉落表 |
| 技能領悟 | 被敵人技能擊中後 5% 機率領悟（被動學習） |

## 8.10 戰鬥失敗

- 主角 HP <= 0 時戰鬥失敗
- 使魔 HP <= 0 時使魔退場，主角繼續戰鬥
- 戰鬥失敗 -> 觸發死亡懲罰（見死亡與懲罰機制）
