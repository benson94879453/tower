# Boss設計

## 15.1 Boss設計原則

- 每個Boss有**獨特機制**，需要特定策略應對
- Boss戰**不可逃跑**
- Boss有多階段行為切換
- 首次擊敗Boss有**保底獎勵**（稀有卷軸/飾品/使魔核心）
- Boss可在競技場重新挑戰（50F解鎖後）

## 15.2 十大區域Boss詳細設計

### Boss 1：石像守衛（9F）

```
元素：無
HP: 500    M.ATK: 25    M.DEF: 30    速度: 10
特色：教學Boss

行為模式：
  回合1-2：普通攻擊（威力30）
  回合3：蓄力（本回合不行動，下回合攻擊x2）
  回合4：重擊（威力60 x2 = 120）
  循環以上模式

教學目的：讓玩家學會「看敵人蓄力 -> 使用防禦/護盾技能」

首次擊敗獎勵：
  - 魔法傀儡使魔核心 x1（100%）
  - 強化石 x3
  - 500G
```

### Boss 2：熔岩巨蛇（19F）

```
元素：火
HP: 1200   M.ATK: 55   M.DEF: 30   速度: 18
特色：持續場地傷害

行為模式：
  每回合：場地燃燒（全場每回合損失 5% HP）
  HP>70%：火焰噴射（單體，威力70，30%燃燒）
  HP 40~70%：熔岩吐息（全體，威力50，50%燃燒）
  HP<40%：狂暴（攻擊力+50%），熔岩噴發（全體，威力80）

弱點策略：水/冰系技能效果絕佳，高回復build可磨死

首次擊敗獎勵：
  - 火焰小鬼使魔核心 x1（100%）
  - 熔岩核心 x1
  - 火系稀有卷軸（隨機）
  - 1000G
```

### Boss 3：冰霜女巫（29F）

```
元素：水
HP: 1500   M.ATK: 70   M.DEF: 45   速度: 25
特色：封印玩家技能

行為模式：
  回合1：冰封詠唱（隨機封印玩家1個技能格，3回合）
  HP>50%：冰錐術（單體威力80）+ 寒冰護盾（減傷30%，2回合）
  HP<50%：深度凍結（封印玩家2個技能格）+ 暴風雪（全體威力60，20%凍結）
  每5回合：絕對零度（單體，30%即死，70%無效）

弱點策略：火系/雷系攻擊，攜帶多元素技能避免被封印後無技可用

首次擊敗獎勵：
  - 冰晶護衛使魔核心 x1（100%）
  - 冰之結晶 x5
  - 水系史詩卷軸
  - 1500G
```

### Boss 4：風暴飛龍（39F）

```
元素：風
HP: 2000   M.ATK: 80   M.DEF: 35   速度: 45（極高）
特色：極高迴避率

行為模式：
  被動：迴避率 50%（大部分攻擊打不中）
  HP>60%：風刃（單體威力90）+ 疾風步（速度再+20%）
  HP 30~60%：龍捲風（全體威力70）+ 風障壁（迴避率再+20%）
  HP<30%：俯衝攻擊（迴避率歸零，單體威力200，之後2回合無法行動）

弱點策略：必中技能、命中率提升buff/飾品。HP<30%時趁俯衝後2回合猛攻

首次擊敗獎勵：
  - 風之使者使魔核心 x1（100%）
  - 龍鱗 x3
  - 風系史詩卷軸
  - 2000G
```

### Boss 5：大地巨像（49F）

```
元素：土
HP: 3500   M.ATK: 60   M.DEF: 100   P.DEF: 120   速度: 8（極慢）
特色：超高防禦

行為模式：
  被動：物理傷害減半
  HP>50%：地震（全體威力60）+ 岩石護甲（防禦再+30%，3回合）
  HP<50%：剝離（移除玩家所有增益buff）+ 落石（單體威力120）
  每6回合：大地之怒（全體威力100 + 速度下降）

弱點策略：DoT（燃燒/中毒）無視防禦。破防技能。風系效果絕佳

首次擊敗獎勵：
  - 岩石守衛使魔核心 x1（100%）
  - 秘銀礦 x5
  - 土系史詩卷軸
  - 2500G
```

### Boss 6：雷神雙子（59F）

```
元素：雷
HP: 各 2000（共4000）  M.ATK: 各75  速度: 各35
特色：雙Boss，需同時處理

行為模式：
  雷神兄：雷擊（單體威力100，30%麻痺）
  雷神弟：充能（為兄累積能量）
  兄被充能後：雷神之怒（全體威力120，50%麻痺）
  一方被擊敗時：另一方進入「覺醒」狀態
    - 攻擊力x2，速度x1.5
    - 每回合自動回復 10% HP
    - 3回合後復活被擊敗的一方（50% HP）

弱點策略：AOE技能同時消耗雙方HP，盡量同回合擊殺兩隻

首次擊敗獎勵：
  - 雷光精靈使魔核心 x1（100%）
  - 雷之結晶 x8
  - 雷系史詩卷軸
  - 稀有飾品（隨機）
  - 3000G
```

### Boss 7：暗影領主（69F）

```
元素：暗
HP: 3000   M.ATK: 90   M.DEF: 60   速度: 30
特色：複製玩家技能

行為模式：
  回合1：暗影複製（掃描玩家技能組，之後可使用玩家的所有已裝備技能）
  HP>50%：使用複製的玩家技能（AI選擇最優組合）
  HP<50%：暗影分裂（召喚2個分身，分身HP500，也會用複製技能）
  HP<20%：暗影吞噬（吸收分身回復HP + 全體暗系威力150）

弱點策略：裝備Boss用起來不利的技能組合（如自傷技能、對暗無效的技能）
          或用光系技能快速擊破

首次擊敗獎勵：
  - 影子刺客使魔核心 x1（100%）
  - 暗之結晶 x8
  - 暗系傳說卷軸
  - 3500G
```

### Boss 8：聖騎士長（79F）

```
元素：光
HP: 4000   M.ATK: 85   M.DEF: 80   P.DEF: 80   速度: 25
特色：魔法反射

行為模式：
  回合1：聖光護盾（反射所有魔法攻擊，持續3回合）
  護盾期間：聖光斬（物理攻擊，威力100）+ 審判（光系，威力80，對暗x3）
  護盾結束後1回合：詠唱（不行動）
  詠唱後：重新施放聖光護盾
  HP<30%：最終審判（全體光系威力200，2回合冷卻）

弱點策略：護盾期間用道具/使魔物理攻擊/等待。護盾結束的1回合詠唱空檔全力輸出

首次擊敗獎勵：
  - 光之精靈使魔核心 x1（100%）
  - 光之結晶 x10
  - 光系傳說卷軸
  - 史詩法袍（聖騎士之袍）
  - 4000G
```

### Boss 9：虛空吞噬者（89F）

```
元素：無
HP: 5000   M.ATK: 100   M.DEF: 70   速度: 35
特色：封印元素

行為模式：
  每2回合：元素吞噬（隨機封印玩家一個元素，該元素所有技能不可用，直到戰鬥結束）
  被動：已吞噬的元素，Boss獲得該元素抗性（-50%傷害）
  HP>50%：虛空射線（無屬性，威力100）+ 次元斬（無屬性，威力80）
  HP<50%：加速吞噬（每1回合封印一個元素）
  HP<20%：虛空崩塌（全體無屬性威力180 + 移除所有buff）

弱點策略：攜帶多元素技能。無屬性技能不會被封印。速戰速決避免被封太多

首次擊敗獎勵：
  - 虛空觀測者使魔核心 x1（100%）
  - 古代碎片 x5
  - 傳說飾品「不屈意志」
  - 5000G
```

### Boss 10：塔之守護者 — 亞克圖斯的分身（99F）

```
元素：全（每階段變化）
HP: 8000   M.ATK: 120   M.DEF: 80   速度: 40
特色：多階段 + 全元素

階段1（HP 100~75%）：火階段
  - 使用強力火系技能
  - 場地燃燒
  
階段2（HP 75~50%）：水+雷階段
  - 切換為水雷複合攻擊
  - 凍結+麻痺組合
  
階段3（HP 50~25%）：暗階段
  - 複製玩家1個技能使用
  - 詛咒+封印
  
階段4（HP 25~0%）：光+全階段
  - 全元素隨機攻擊
  - 每回合回復 3% HP
  - 最終技：根源之力（全體威力250，3回合冷卻）

首次擊敗獎勵：
  - 混沌幼體使魔核心 x1（100%）
  - 全元素結晶 x5
  - 傳說法杖「守護者之杖」
  - 10000G
```

### 最終Boss：塔之核心（100F）

```
元素：無/全
HP: 12000
特色：三階段戰鬥 + 結局選擇

第一階段：物質之殼（HP 4000）
  - 高防禦+物理攻擊
  - 需要魔法攻擊破殼

第二階段：魔力之核（HP 4000）
  - 反射魔法+魔法攻擊
  - 需要物理手段或無屬性攻擊

第三階段：根源意志（HP 4000）
  - 每回合切換弱點元素（只有當回合弱點元素能造成正常傷害，其餘x0.1）
  - 弱點元素會在畫面上提示
  - 根源之力（全體威力300，5回合冷卻）

擊敗後觸發結局選擇

首次擊敗獎勵：
  - 傳說法杖「亞克圖斯之杖」
  - 傳說法袍「星辰法袍」
  - 傳說飾品「根源寶珠」（全屬性+10%，全抗性+10%）
  - 通關成就
  - 解鎖二周目（敵人強化，新增隱藏內容）
```
