# 數值公式與數值平衡

## C.1 傷害計算公式

### 基礎傷害公式

```
基礎傷害 = (技能威力 x 攻擊方M.ATK / 防禦方M.DEF) x 屬性倍率 x 隨機修正

其中：
  攻擊方M.ATK = 基礎M.ATK + 裝備加成 + buff加成
  防禦方M.DEF = 基礎M.DEF + 裝備加成 + buff加成
  屬性倍率 = 查表（見元素相剋表）
  隨機修正 = 0.85 ~ 1.15（均勻分布）
```

### 暴擊傷害

```
暴擊判定 = rand() < 暴擊率
暴擊傷害 = 基礎傷害 x 1.5（預設暴擊倍率）

暴擊率 = 基礎暴擊率 + 裝備加成 + buff加成
暴擊率上限 = 80%
```

### 物理傷害公式

```
物理傷害 = (技能威力 x 攻擊方P.ATK / 防禦方P.DEF) x 隨機修正
（不受屬性倍率影響）
```

### 命中判定

```
命中率 = 技能命中率 x (攻擊方命中 / (攻擊方命中 + 防禦方迴避))

命中率下限 = 10%（除特殊技能外，總有最低命中率）
必中技能：命中率 = 100%（無視迴避）
```

## C.2 屬性成長公式

### 主角升級屬性

```
HP(Lv)  = 100 + (Lv - 1) x 12
MP(Lv)  = 60  + (Lv - 1) x 8
M.ATK(Lv) = 15 + (Lv - 1) x 3
M.DEF(Lv) = 10 + (Lv - 1) x 2
P.ATK(Lv) = 8  + (Lv - 1) x 1
P.DEF(Lv) = 8  + (Lv - 1) x 1.5
速度(Lv)  = 15 + (Lv - 1) x 1.5
命中(Lv)  = 100 + (Lv - 1) x 0.5
迴避(Lv)  = 5  + (Lv - 1) x 0.3

範例（Lv.50）：
HP  = 100 + 49 x 12 = 688
MP  = 60  + 49 x 8  = 452
M.ATK = 15 + 49 x 3 = 162
M.DEF = 10 + 49 x 2 = 108
```

### 經驗值需求

```
所需EXP(Lv) = 50 + (Lv^2) x 5

Lv.1→2:  55
Lv.10→11: 550
Lv.50→51: 12,550
Lv.99→MAX: 49,055
```

### 使魔升級經驗

```
使魔所需EXP(Lv) = 30 + (Lv^2) x 3
使魔獲得EXP = 主角獲得EXP x 0.5
```

## C.3 裝備強化公式

### 強化成功率

```
成功率(+N) = max(10%, 100% - N x 10%)

+0 → +1: 100%
+1 → +2: 90%
+2 → +3: 80%
+3 → +4: 70%
+4 → +5: 60%
+5 → +6: 50%
+6 → +7: 40%
+7 → +8: 30%
+8 → +9: 20%
+9 → +10: 10%
```

### 強化數值加成

```
法杖強化：
  M.ATK加成(+N) = 基礎M.ATK x (N x 0.08)
  即每+1 = 基礎攻擊力的 8%

法袍強化：
  M.DEF加成(+N) = 基礎M.DEF x (N x 0.08)
  P.DEF加成(+N) = 基礎P.DEF x (N x 0.08)
  HP加成(+N)    = 基礎HP加成 x (N x 0.05)
```

### 強化費用

```
強化費用(+N) = 基礎費用 x (1 + N x 0.5)

普通裝備基礎費用: 100G
稀有裝備基礎費用: 300G
史詩裝備基礎費用: 800G
傳說裝備基礎費用: 2000G
```

## C.4 經濟平衡

### 金幣收入

```
戰鬥金幣 = 基礎金幣 x (1 + 樓層 / 50) x 難度修正
Boss金幣 = 基礎金幣 x 10

每區域預期金幣收入/層（普通戰鬥平均）：
1F~9F:    15G/戰
11F~19F:  25G/戰
21F~29F:  40G/戰
31F~39F:  55G/戰
41F~49F:  75G/戰
51F~59F:  100G/戰
61F~69F:  130G/戰
71F~79F:  170G/戰
81F~89F:  220G/戰
91F~99F:  300G/戰
```

### 金幣支出

```
商品價格隨樓層調整：
  商品價格 = 基礎價格 x (1 + 解鎖安全區層數 / 100)

每區域預期支出（回復藥+強化+技能研究）：
  設計目標：收入略大於支出（約 1.2 倍）
  確保玩家不會因正常遊玩而破產
  死亡懲罰（20%金幣）不應造成惡性循環
```

## C.5 掉落率公式

```
最終掉落率 = 基礎掉落率 x (1 + 幸運加成) x 稀有度修正

幸運加成來源：
  - 幸運貓使魔：+20%
  - 掉落率飾品：+10%~+30%
  - 稱號加成：+10%~+20%
  - 祭壇祝福：+15%（臨時）

稀有度修正（對應稀有度的額外乘數）：
  N:   x1.0
  R:   x0.8
  SR:  x0.5
  SSR: x0.3
  UR:  x0.1
```

## C.6 敵人數值曲線

```
敵人HP(樓層) = 80 + 樓層 x 25 + (樓層/10)^2 x 10
敵人M.ATK(樓層) = 15 + 樓層 x 2.5
敵人M.DEF(樓層) = 8 + 樓層 x 1.5
敵人速度(樓層) = 10 + 樓層 x 0.8

精英修正 = 基礎 x 2.0
Boss修正 = 基礎 x 4.0 ~ 8.0（根據具體Boss設計）

範例（50F 普通敵人）：
HP     = 80 + 50x25 + 25x10 = 1580
M.ATK  = 15 + 50x2.5 = 140
M.DEF  = 8 + 50x1.5 = 83
速度   = 10 + 50x0.8 = 50
```

## C.8 技能熟練度需求

```
升級所需熟練度：
  Lv.1 → Lv.2: 100 使用次數
  Lv.2 → Lv.3: 250 使用次數
  Lv.3 → Lv.MAX: 500 使用次數

突變所需熟練度：500 使用次數 + 特殊素材

技能每次使用 = +1 熟練度
技能弱點命中 = +2 熟練度
技能擊敗敵人 = +3 熟練度
```

## C.9 二周目數值修正

通關後解鎖的二周目（強力新遊戲）：

```
二周目修正：
  敵人HP: x1.5
  敵人攻擊: x1.3
  敵人防禦: x1.3
  敵人速度: x1.2
  Boss特殊: 新增額外行動模式
  掉落率: x1.5（補償難度）
  EXP: x1.3
  Gold: x1.3

二周目新增內容：
  - 每區域新增1種隱藏敵人
  - 新增5個隱藏Boss（非區域Boss）
  - 新增隱藏結局條件
  - 解鎖UR稀有度技能卷軸掉落
```

## C.10 平衡設計目標

```
核心平衡原則：
  1. 首次通關建議時間：30~40小時
  2. 100%完成度（全技能/全使魔/全成就）：80~100小時
  3. 每個區域通過時間（含練級）：3~4小時
  4. Boss戰預期回合數：8~15回合
  5. 普通戰鬥預期回合數：3~5回合
  6. 精英戰鬥預期回合數：5~8回合
  7. 死亡後回復進度所需時間：<30分鐘
  8. 強化裝備到+5的預期花費：約1小時收入
  9. 強化裝備到+10的預期花費：約8小時收入
  10. 每個元素至少有1個可行的通關build
```
