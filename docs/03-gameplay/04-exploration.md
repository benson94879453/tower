# 節點式地牢探索系統

## 7.1 設計概要

地牢探索**完全透過點擊選項**進行，不使用格子地圖或即時移動。每層地牢是一個**節點路線圖**，玩家點擊選擇前進方向與互動方式。

## 7.2 節點地圖生成規則

每個非安全區樓層自動生成一張節點地圖：

```
[入口]
  |
  +--- [戰鬥] ---+--- [事件] ---- [戰鬥] ---+
  |              |                          |
  +--- [寶箱] ---+--- [戰鬥] --- [商人] ----+
                                            |
                                        [中繼點]
                                            |
                           +--- [精英戰] ---+
                           |                |
                           +--- [事件] ---- [祭壇] ---+
                                                      |
                                                   [出口]
```

**生成規則：**
- 地圖分為 **3~4 行**（階段）
- 每行有 **2~3 個節點**
- 相鄰行的節點之間隨機連線（保證每個節點至少有一條路可達）
- 入口和出口各一個，固定在首尾
- 節點類型按權重隨機分配

**路線選擇要素：**
- 玩家可以看到節點類型圖示（但不知道具體內容）
- 部分節點在暗屬性區域顯示為「???」
- 選擇路線 = 選擇風險與報酬的平衡

## 7.3 節點類型詳細設計

### 7.3.1 戰鬥節點

進入後觸發戰鬥遭遇。

**戰鬥後：**
- 勝利 -> 獲得經驗值、金幣、隨機掉落物
- 逃跑成功 -> 無獲得，回到節點地圖
- 逃跑失敗 -> 被偷襲（敵人先手 1 回合）後正常戰鬥

### 7.3.2 事件節點

隨機觸發各類事件，透過選項互動。

**事件類型：**

| 類型 | 頻率 | 說明 |
|------|------|------|
| 發現類 | 30% | 發現寶物、魔法陣、前人遺跡 |
| 遭遇類 | 20% | 遇到NPC、迷路的冒險者、神秘生物 |
| 陷阱類 | 15% | 魔法陷阱、地板機關、詛咒之門 |
| 謎題類 | 15% | 符文解讀、魔法機關、邏輯謎題 |
| 環境類 | 10% | 特殊地形、元素湧泉、魔力結晶 |
| 劇情類 | 10% | 導師筆記、塔的記憶、主線相關 |

**技能/道具解鎖額外選項：**
玩家持有特定技能或道具時，會出現額外的選項。

### 7.3.3 精英戰節點

- **顯示方式**：節點地圖上以特殊圖示標記
- **敵人特點**：比同層普通敵人等級高 5 級，有獨特技能
- **獎勵**：保證掉落一件稀有道具或卷軸
- **可選擇繞過**（如果路線允許）

### 7.3.4 寶箱節點

**寶箱品質隨樓層提升：**
- 1F~20F：普通/優良 寶箱
- 21F~50F：優良/稀有 寶箱
- 51F~80F：稀有/史詩 寶箱
- 81F~100F：史詩/傳說 寶箱

### 7.3.5 商人節點

- **商品**：隨機 3~5 件商品，可能包含當前區域買不到的稀有品
- **可討價還價**：選擇「殺價」有 30% 機率成功（降價 20%），失敗則商人提價 10%
- **特殊商人**：極小機率遇到「神秘商人」，販賣傳說級物品但價格極高

### 7.3.6 中繼節點

- **休息**：回復 30% HP/MP
- **整理裝備**：開啟裝備介面
- **研讀魔法書**：提升技能熟練度

### 7.3.7 祭壇節點

高風險高報酬的特殊節點。

| 祭壇 | 獻祭物 | 報酬 |
|------|--------|------|
| 血之祭壇 | 30% 當前HP | 隨機稀有技能卷軸 |
| 魔力祭壇 | 50% 當前MP | 永久MP上限+5 |
| 財富祭壇 | 500G | 隨機稀有飾品 |
| 靈魂祭壇 | 使魔核心x1 | 使魔永久強化 |
| 命運祭壇 | 隨機裝備1件 | 獲得比獻祭品高一級的裝備 |

## 7.4 探索中的資源管理

探索過程中，以下資源持續消耗：

- **HP**：戰鬥消耗、陷阱傷害、事件代價
- **MP**：技能消耗、探索選項消耗
- **道具**：消耗品使用
- **技能PP**：每個技能有使用次數限制

**資源補充方式：**
- 中繼點休息（回復 30%）
- 使用藥水（消耗道具欄）
- 部分事件選項
- 回到安全區（完全回復）

**核心張力**：資源有限 -> 是繼續推進還是回安全區？

## 7.5 傳送卷軸機制

- 在探索中隨時可使用「傳送卷軸」返回最近的安全區
- 傳送後，當前樓層的進度**不會保留**（需要重新挑戰該層）
- Boss層不可使用傳送卷軸（進入Boss戰後必須戰勝或死亡）
