# 死亡與懲罰機制

## 19.1 死亡觸發條件

- 主角 HP <= 0 時觸發死亡
- 使魔退場不算死亡（只是使魔不可用）

## 19.2 死亡懲罰

採用**中度懲罰**設計，有損失但不會讓玩家絕望：

```
死亡後：
  1. 傳送回最近的安全區（已啟動的傳送門）
  2. HP/MP 全回復
  3. 使魔 HP 全回復
  
  懲罰：
  - 失去當前攜帶金幣的 20%
  - 本次探索獲得的未鑑定道具全部消失
  - 已鑑定的道具保留
  - 技能PP全回復（算是好處）
  - 當前樓層進度重置（需重新挑戰）
```

## 19.3 死亡保護機制

有多種方式減輕或避免死亡懲罰：

| 保護方式 | 效果 | 取得 |
|---------|------|------|
| 守護天使飾品 | HP首次歸零時回復30%（每場1次） | 聖光區Boss |
| 靈魂鏈接飾品 | 致命傷由使魔代替承受 | 隱藏任務 |
| 復活卷軸 | 死亡時自動使用，HP回復50% | 80F+商店/掉落 |
| 保險箱（1F） | 存入的金幣不受死亡影響 | 安全區NPC |
| 祝福狀態 | 死亡時保留所有道具（不失去未鑑定品） | 聖光殿堂祭壇 |
