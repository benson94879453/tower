# 安全區與城鎮系統

## 6.1 安全區總覽

每 10 層設置一個安全區，安全區內無敵人、無戰鬥，是玩家的休息與準備據點。

| 樓層 | 名稱 | 規模 | 特色 |
|------|------|------|------|
| **1F** | 黎明之城 | 大型主城 | 全設施。遊戲起點與主要據點 |
| **10F** | 冒險者前哨站 | 小型 | 商店、旅館、簡易鍛造 |
| **20F** | 冰原營地 | 小型 | 商店、旅館、使魔馴養師 |
| **30F** | 風之驛站 | 中型 | 商店、旅館、卷軸專賣商人 |
| **40F** | 地底市集 | 中型 | 商店、旅館、鍛造大師、稀有素材商 |
| **50F** | 中層都市 | 大型（第二主城） | 全設施 + 競技場 + 特殊NPC |
| **60F** | 暗之避難所 | 中型 | 商店、旅館、詛咒解除師、暗系專賣 |
| **70F** | 雷霆要塞 | 中型 | 商店、旅館、附魔師、雷系專賣 |
| **80F** | 聖域 | 中型 | 商店、旅館、高階鍛造、傳說卷軸商 |
| **90F** | 最後營地 | 小型 | 商店、旅館、全buff NPC、最終準備 |

## 6.2 城鎮設施詳細設計

### 6.2.1 魔法商店

- **功能**：購買/販賣 消耗品、基礎裝備、卷軸
- **庫存機制**：
  - 基礎商品固定供應
  - 稀有商品隨機刷新（每次進入安全區時刷新）
  - 高層安全區解鎖更高級商品
- **價格**：隨樓層提升，高層商品更貴但更強

### 6.2.2 鍛造工坊

- **功能**：
  - **裝備製造**：素材 + 金幣 -> 新裝備
  - **裝備強化**：素材 + 金幣 -> 裝備 +1/+2/+3...（最高 +10）
  - **飾品合成**：特定素材組合 -> 飾品
  - **裝備分解**：不需要的裝備 -> 回收素材
- **強化機制**：
  - +1 ~ +5：100% 成功率
  - +6 ~ +7：80% 成功率
  - +8 ~ +9：60% 成功率
  - +10：40% 成功率
  - 失敗不會降級，但素材和金幣消耗

### 6.2.3 使魔小屋

- **功能**：
  - **孵化**：使用「使魔核心」+ 素材 + 金幣孵化新使魔
  - **培養**：餵食素材提升使魔經驗
  - **技能教學**：使用卷軸教使魔新技能
  - **進化**：滿足條件後進化使魔
  - **存放**：存放不攜帶的使魔（最多存放 10 隻）
  - **更換**：切換攜帶的使魔（攜帶上限 1 隻）

### 6.2.4 大圖書館

- **功能**：
  - **技能研究**：消耗「研究點數」+ 素材，合成/研發新魔法
  - **技能突變查詢**：查看已學技能的突變路線與條件
  - **敵人圖鑑**：查看已遭遇敵人的詳細資料（弱點、掉落物）
  - **元素知識**：查看屬性相剋表與詳細倍率
  - **導師筆記**：主線劇情文件收集與閱讀

### 6.2.5 冒險者酒館

- **功能**：
  - **接取任務**：支線任務板，每次刷新 3~5 個任務
  - **任務類型**：
    - 討伐任務：在指定區域擊敗 X 隻特定敵人
    - 收集任務：收集 X 個特定素材
    - 探索任務：到達指定樓層的特定節點
    - 挑戰任務：在限定回合內擊敗特定敵人
  - **任務獎勵**：金幣、素材、稀有卷軸、飾品圖紙
  - **NPC 對話**：與酒館內的冒險者交談，獲取情報或觸發隱藏任務
- **任務上限**：同時最多接取 **3 個任務**

### 6.2.6 旅館

- **功能**：
  - **休息**：完全回復 HP 和 MP
  - **存檔**：手動存檔點（遊戲也有自動存檔）
  - **做夢（隨機事件）**：休息時有 20% 機率觸發夢境事件
    - 夢見未知的魔法 -> 獲得隨機技能線索
    - 夢見導師 -> 主線劇情提示
    - 噩夢 -> 醒來後獲得暫時減益但有特殊道具

### 6.2.7 競技場（50F 中層都市限定）

- **功能**：
  - **挑戰模式**：連續戰鬥不回復，看能撐幾場
  - **Boss 回顧戰**：重新挑戰已擊敗的 Boss（可選難度倍率）
  - **排名**：記錄最佳成績
  - **獎勵**：競技場幣 -> 兌換獨佔飾品/卷軸

### 6.2.8 附魔師（70F 限定）

- **功能**：為裝備附加元素屬性或特殊效果
  - 法杖附火元素 -> 所有攻擊附帶火屬性
  - 法袍附冰抗 -> 冰系傷害減免 20%
  - 附魔可覆蓋，但之前的附魔會消失

### 6.2.9 詛咒解除師（60F 限定）

- **功能**：
  - 解除被詛咒的裝備（詛咒裝備屬性強但有負面效果，解除後保留正面效果）
  - 解除探索中獲得的持續性詛咒狀態
