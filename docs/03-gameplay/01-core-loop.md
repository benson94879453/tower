# 遊戲結構與核心循環

## 4.1 主循環

```
城鎮準備 → 選擇樓層 → 節點探索 → 戰鬥/事件 → 收集戰利品
   ↑                                                ↓
   └──── 回到安全區（自願或死亡）←── 打Boss上樓 ←──┘
```

## 4.2 單層循環

```
進入樓層 → 查看節點地圖 → 選擇路線 → 逐節點推進 → Boss戰 → 上樓
```

## 4.3 單節點循環

```
進入節點 → 閱讀描述 → 查看選項 → 做出選擇 → 獲得結果 → 回到節點地圖
```

## 4.4 戰鬥循環

```
遭遇敵人 → 選擇行動（技能/道具/使魔/逃跑）→ 
  雙方結算（速度決定先後）→ 狀態結算 → 
  判斷勝負 → 戰利品/經驗值
```

## 4.5 遊戲時間預估

| 內容 | 預估時間 |
|------|---------|
| 首次通關（100F） | 30~50 小時 |
| 全技能收集 | 80+ 小時 |
| 全使魔培養 | 60+ 小時 |
| 全成就 | 100+ 小時 |
