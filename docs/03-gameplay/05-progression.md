# 傳送與進度系統

## 18.1 傳送門機制

- 每次擊敗Boss後，該Boss層的傳送門**永久啟動**
- 已啟動的傳送門可以在任意安全區的傳送陣使用
- 傳送門允許直接傳送到對應安全區（跳過中間樓層）

## 18.2 樓層進度

- 每個樓層的通關狀態獨立記錄
- 通過一層（到達出口）= 解鎖下一層的入口
- Boss層需要擊敗Boss才算通過
- **樓層不會變化**：每次重新挑戰同一層，節點地圖會重新生成，但敵人等級和類型範圍不變

## 18.3 存檔系統

- **自動存檔**：每次進入/離開安全區時自動存檔
- **手動存檔**：在安全區可隨時手動存檔
- **存檔欄位**：3 個存檔欄
- **探索中不可存檔**（Roguelike 特性）
- 探索中退出遊戲 = 進度丟失，回到最近安全區
