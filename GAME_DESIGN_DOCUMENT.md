## 1. 遊戲概述

### 1.1 一句話描述

> 以魔法師身份攀登百層高塔，透過收集魔法卷軸與培養使魔，以回合制戰鬥征服每一層的冒險RPG。

### 1.2 遊戲類型

- **主類型**：回合制 RPG
- **次類型**：Roguelite（程序生成樓層內容 + 永久進度保留）
- **靈感來源**：
  - 寶可夢 1~9 代（回合制技能戰鬥、屬性相剋、大量技能池）
  - Sword Art Online S1（100層高塔、安全區城鎮、層層攻略）
  - Slay the Spire（節點式地圖選擇、build構築）
  - 不思議迷宮（點擊式探索、事件選項）

### 1.3 目標玩家

- 喜歡回合制策略戰鬥的玩家
- 喜歡收集、培養、構築 build 的玩家
- 喜歡逐步攻略、有明確進度目標的玩家
- 不擅長或不喜歡動作操作的玩家

### 1.4 核心賣點

1. **龐大的技能池**（200+ 魔法技能）構築無限 build 組合
2. **100 層高塔**提供明確的長期目標感
3. **使魔培養**增加戰鬥夥伴養成樂趣
4. **純點擊操作**，所有互動透過選項完成，零操作門檻
5. **幾何極簡美術風格**，用 UI 品質取代美術素材

### 1.5 操作方式

- **僅使用滑鼠點擊**
- 所有探索、戰鬥、城鎮互動皆透過畫面上的按鈕/選項完成
- 支援鍵盤快捷鍵（數字鍵 1~6 對應技能格）作為輔助

## 2. 核心設計理念

### 2.1 決策驅動

> 遊戲的每一刻都在做有意義的選擇

- 探索時：選擇走哪條路線、如何處理事件
- 戰鬥時：選擇使用哪個技能、何時用道具、何時逃跑
- 成長時：選擇學哪個技能、帶哪隻使魔、裝哪個飾品

### 2.2 Build 構築

> 每次攀塔都能嘗試不同的流派組合

- 技能格有限（6主動+2被動），逼玩家專精
- 飾品（3格）定義核心流派
- 使魔類型補強或互補
- 沒有「最優解」，鼓勵實驗

### 2.3 風險與報酬

> 越危險的選擇，越豐厚的回報

- 精英戰難但掉落稀有
- 事件中大膽的選項可能大賺或大虧
- PP/MP 資源管理：強技能消耗大，何時用是關鍵
- 繼續推進 vs 回安全區補給的抉擇

### 2.4 漸進式複雜度

> 前 10 層教會基礎，後 90 層逐步加深

- 1~9F：只有基礎戰鬥和簡單事件
- 10~30F：引入飾品、使魔、屬性相剋
- 30~60F：引入技能突變、精英戰、複合屬性敵人
- 60~100F：全系統開放，高難度挑戰

## 3. 世界觀與故事背景

### 3.1 世界設定

遠古時代，大魔導師「亞克圖斯」建造了一座通天高塔——**魔導之塔**。塔內封印了無數魔法知識與遠古力量。數百年來，無數魔法師前往挑戰高塔，試圖到達塔頂獲取「根源之力」——據說能實現任何願望的終極魔法。

然而，至今無人成功登頂。塔內充滿了失控的魔法造物、被封印的遠古生物、以及前人留下的陷阱與遺產。

### 3.2 主角設定

- **身份**：剛從皇家魔法學院畢業的年輕魔法師
- **動機**：為了尋找失蹤的導師（導師數年前進入高塔後失聯）
- **特點**：擅長快速學習魔法（遊戲機制：能從卷軸/書學技能）
- **外觀**：幾何風格的法師剪影——圓形頭部 + 三角形帽子 + 梯形身體 + 法杖

### 3.3 高塔的秘密（主線劇情）

隨著攀塔逐漸揭露的真相：

| 進度 | 揭露內容 |
|------|---------|
| 10F | 發現導師留下的筆記，得知塔內有「守護者」系統 |
| 20F | 遇到其他攻略者的幽魂，得知塔會根據挑戰者調整難度 |
| 30F | 發現導師曾到達 50F，留下重要研究資料 |
| 50F | 在中層都市發現導師的研究室，得知塔的真正目的 |
| 70F | 發現亞克圖斯並未死亡，而是成為了塔的一部分 |
| 90F | 最後營地，導師的最終留言 |
| 100F | 面對塔之核心，選擇結局 |

### 3.4 結局分支

- **結局 A：繼承**——接管高塔的力量，成為新的守護者
- **結局 B：解放**——摧毀塔之核心，釋放所有封印的靈魂
- **結局 C：超越**——找到第三條路，與導師一同重建高塔的秩序

## 4. 遊戲結構與核心循環

### 4.1 主循環

```
城鎮準備 → 選擇樓層 → 節點探索 → 戰鬥/事件 → 收集戰利品
   ↑                                                ↓
   └──── 回到安全區（自願或死亡）←── 打Boss上樓 ←──┘
```

### 4.2 單層循環

```
進入樓層 → 查看節點地圖 → 選擇路線 → 逐節點推進 → Boss戰 → 上樓
```

### 4.3 單節點循環

```
進入節點 → 閱讀描述 → 查看選項 → 做出選擇 → 獲得結果 → 回到節點地圖
```

### 4.4 戰鬥循環

```
遭遇敵人 → 選擇行動（技能/道具/使魔/逃跑）→ 
  雙方結算（速度決定先後）→ 狀態結算 → 
  判斷勝負 → 戰利品/經驗值
```

### 4.5 遊戲時間預估

| 內容 | 預估時間 |
|------|---------|
| 首次通關（100F） | 30~50 小時 |
| 全技能收集 | 80+ 小時 |
| 全使魔培養 | 60+ 小時 |
| 全成就 | 100+ 小時 |

## 5. 高塔結構（100層）

### 5.1 總體結構

魔導之塔共 100 層，分為 **10 個區域**，每區域 10 層。每區域有獨特的元素主題、視覺風格、敵人種類與環境機制。

```
100F ── 最終Boss：塔頂・封印之間
 ⋮
91F~99F ── 第十區：混沌領域（全屬性混合）
90F ── 安全區：最後營地
 ⋮
81F~89F ── 第九區：虛空之境（無屬性特殊區）
80F ── 安全區：聖域
 ⋮
71F~79F ── 第八區：聖光殿堂（光屬性）
70F ── 安全區：雷霆要塞
 ⋮
61F~69F ── 第七區：暗影深淵（暗屬性）
60F ── 安全區：暗之避難所
 ⋮
51F~59F ── 第六區：雷鳴雲海（雷屬性）
50F ── 安全區：中層都市（第二主城）
 ⋮
41F~49F ── 第五區：大地迷宮（土屬性）
40F ── 安全區：地底市集
 ⋮
31F~39F ── 第四區：暴風迴廊（風屬性）
30F ── 安全區：風之驛站
 ⋮
21F~29F ── 第三區：極寒冰域（水/冰屬性）
20F ── 安全區：冰原營地
 ⋮
11F~19F ── 第二區：熔岩鍛爐（火屬性）
10F ── 安全區：冒險者前哨站
 ⋮
2F~9F  ── 第一區：廢墟入口（無屬性・教學區）
1F ── 起始城鎮：黎明之城（主城）
```

### 5.2 十大區域詳細設計

#### 第一區：廢墟入口（1F~9F）

- **元素**：無屬性
- **環境描述**：高塔底層的廢棄遺跡，佈滿灰塵的石牆與殘破的魔法陣。偶爾能看到前人留下的塗鴉與記號。
- **特色機制**：教學區，逐步引導玩家熟悉戰鬥、探索、道具使用。
- **敵人類型**：魔法造物（石像兵、魔法傀儡）、低階魔獸（史萊姆、蝙蝠）
- **環境事件**：殘破的魔法陣（修復可獲得獎勵）、前人的筆記（提供遊戲提示）
- **Boss（9F）**：石像守衛——教學Boss，行動模式固定，讓玩家學會看敵人招式

#### 第二區：熔岩鍛爐（11F~19F）

- **元素**：火
- **環境描述**：熾熱的熔岩從牆壁裂縫中流出，空氣扭曲顫動。古代鍛造設施依然運作，不斷產出火焰造物。
- **特色機制**：「灼熱環境」——每進入一個戰鬥節點，戰鬥開始時場地自帶「灼熱」狀態（非火系技能威力-10%）
- **敵人類型**：火焰精靈、熔岩蜥蜴、鍛造傀儡、火蛇
- **環境事件**：古代鍛爐（可用素材鍛造裝備）、岩漿河（需要冰系技能或特殊道具渡過）
- **Boss（19F）**：熔岩巨蛇——每回合施加場地燃燒效果，需要水/冰系技能或高回復來對抗

#### 第三區：極寒冰域（21F~29F）

- **元素**：水/冰
- **環境描述**：被永久凍結的樓層，冰晶從天花板垂落，呼出的氣息瞬間凝結。靜謐而危險。
- **特色機制**：「極寒環境」——每 3 回合，所有角色（含敵人）有 15% 機率被凍結 1 回合
- **敵人類型**：冰晶精靈、雪狼、冰封亡靈、寒霜巫師
- **環境事件**：冰封的寶箱（需火系技能解凍）、冰壁後的隱藏通道
- **Boss（29F）**：冰霜女巫——能凍結玩家的技能格（隨機封印 1~2 個技能 3 回合），需要解封手段或靈活應對

#### 第四區：暴風迴廊（31F~39F）

- **元素**：風
- **環境描述**：環繞高塔外側的開放迴廊，狂風呼嘯不止。雲層就在腳下，視野開闊卻步步驚心。
- **特色機制**：「強風環境」——所有遠程魔法命中率 -15%，但風系技能傷害 +20%
- **敵人類型**：風暴鷹、氣旋精靈、風刃獵手、雷雲獸
- **環境事件**：斷橋（需風系技能架設氣流橋）、風之神殿碎片（收集可解鎖特殊風系技能）
- **Boss（39F）**：風暴飛龍——極高迴避率（50%），需命中率提升手段或必中技能

#### 第五區：大地迷宮（41F~49F）

- **元素**：土
- **環境描述**：錯綜複雜的地底迷宮，厚重的岩壁上嵌著發光的礦石。空間不斷緩慢變化。
- **特色機制**：「迷宮效應」——節點地圖上部分路徑會隨機變化（每次進入此區域路線不同）
- **敵人類型**：岩石巨人、地穴蜘蛛、礦石傀儡、地底蠕蟲
- **環境事件**：礦石採集點（獲得鍛造素材）、地震陷阱（躲避檢定）、隱藏的地底湖
- **Boss（49F）**：大地巨像——超高防禦（物理防禦 300%），需要破防技能、持續傷害（DoT）或特殊穿透效果

#### 第六區：雷鳴雲海（51F~59F）

- **元素**：雷
- **環境描述**：高塔深入雲層之上，閃電不斷在雲海中穿梭。空氣中充滿靜電。
- **特色機制**：「帶電環境」——每回合結束有 10% 機率產生隨機落雷，對場上一方造成固定傷害
- **敵人類型**：雷電精靈、電磁傀儡、風暴元素、雷獸
- **環境事件**：避雷石（消耗可免疫一次落雷）、充能裝置（消耗 MP 換取增益）
- **Boss（59F）**：雷神雙子——兩個Boss同時出現，需要 AOE 或同回合擊殺（一方死亡時另一方會狂暴+全回復）

#### 第七區：暗影深淵（61F~69F）

- **元素**：暗
- **環境描述**：光線無法穿透的黑暗空間，只有魔法光源能照亮周圍。低語聲從四面八方傳來。
- **特色機制**：「黑暗環境」——探索時部分節點內容被隱藏（顯示為「???」），需要光系技能或道具照明才能看到
- **敵人類型**：暗影刺客、深淵蠕蟲、夜魔、噩夢具現體
- **環境事件**：暗影祭壇（獻祭 HP 換取暗系技能書）、迷失者的低語（心智檢定，失敗則混亂）
- **Boss（69F）**：暗影領主——複製玩家的技能組來攻擊。玩家裝備什麼技能，Boss 就會用同樣的技能。需要準備 Boss 用起來不利的技能組合

#### 第八區：聖光殿堂（71F~79F）

- **元素**：光
- **環境描述**：被神聖光芒籠罩的殿堂，一切黑暗無所遁形。壯麗的彩繪窗與光柱令人敬畏。
- **特色機制**：「聖光環境」——所有回復效果 +30%，但暗系技能傷害 -50%
- **敵人類型**：聖光傀儡、光之守衛、審判天使、光輝獸
- **環境事件**：祈禱祭壇（消耗金幣獲得祝福buff）、審判之門（回答謎題通過或戰鬥通過）
- **Boss（79F）**：聖騎士長——能反射所有魔法攻擊 3 回合（反射護盾），需要等待或用非魔法手段（道具、使魔物理攻擊）

#### 第九區：虛空之境（81F~89F）

- **元素**：無（特殊）
- **環境描述**：超越常理的空間，法則扭曲。上下左右的概念模糊，空間碎片漂浮。
- **特色機制**：「法則扭曲」——每場戰鬥開始時隨機改變一條規則（例：屬性相剋反轉、速度最慢先出手、回復變傷害等）
- **敵人類型**：虛空行者、次元裂縫、概念具現體、法則守護者
- **環境事件**：次元裂縫（進入隨機獲得增益或減益）、時間停滯區（可回溯上一次選擇）
- **Boss（89F）**：虛空吞噬者——每 2 回合隨機封印玩家一個元素（該元素所有技能不可用），需要多元素 build 或無屬性技能

#### 第十區：混沌領域（91F~99F）

- **元素**：全屬性混合
- **環境描述**：所有元素在此交匯碰撞，空間不斷崩塌重組。這是高塔最接近核心的區域。
- **特色機制**：「元素風暴」——每回合隨機一個元素被強化（傷害x2），下回合換另一個。需要見機行事
- **敵人類型**：混沌元素、元素融合體、塔之守護者、前代挑戰者亡魂
- **環境事件**：元素漩渦（可吸收強化一個技能，但有風險）、亞克圖斯的記憶碎片（主線劇情）
- **Boss（99F）**：塔之守護者（亞克圖斯的分身）——使用所有 8 元素技能，階段制（每損失 25% HP 換一個階段，每階段主用不同元素）

#### 最終層（100F）

- **塔之核心**——最終Boss，多階段戰鬥
- 詳見 [15. Boss設計](#15-boss設計) 100F 部分

### 5.3 每層內部結構

每個非安全區樓層包含以下固定結構：

| 元素 | 數量 | 說明 |
|------|------|------|
| 入口節點 | 1 | 從樓梯進入 |
| 戰鬥節點 | 3~6 | 隨機敵人遭遇 |
| 事件節點 | 1~3 | 隨機事件 |
| 精英節點 | 0~1 | 強敵（非每層都有） |
| 寶箱節點 | 1~2 | 隨機獎勵 |
| 商人節點 | 0~1 | 移動商人（非每層都有） |
| 中繼節點 | 0~1 | 回復點（非每層都有） |
| 祭壇節點 | 0~1 | 特殊獻祭（稀少） |
| 出口/Boss | 1 | 前往下一層（x9F 層為 Boss） |

每層總節點數：**8~15 個**，組成 2~3 條路線可選擇。

### 5.4 樓層難度曲線

```
敵人等級 ≈ 樓層數 x 1.0 + 區域修正
敵人HP   ≈ 基礎值 x (1 + 樓層數 x 0.08)
敵人攻擊 ≈ 基礎值 x (1 + 樓層數 x 0.06)
經驗值   ≈ 基礎值 x (1 + 樓層數 x 0.05)
掉落品質 ≈ 隨樓層提升，高層出現稀有/傳說級道具
```

### 6.2 城鎮設施詳細設計

#### 6.2.1 魔法商店

- **功能**：購買/販賣 消耗品、基礎裝備、卷軸
- **庫存機制**：
  - 基礎商品固定供應
  - 稀有商品隨機刷新（每次進入安全區時刷新）
  - 高層安全區解鎖更高級商品
- **價格**：隨樓層提升，高層商品更貴但更強

```

#### 6.2.2 鍛造工坊

- **功能**：
  - **裝備製造**：素材 + 金幣 -> 新裝備
  - **裝備強化**：素材 + 金幣 -> 裝備 +1/+2/+3...（最高 +10）
  - **飾品合成**：特定素材組合 -> 飾品
  - **裝備分解**：不需要的裝備 -> 回收素材
- **強化機制**：
  - +1 ~ +5：100% 成功率
  - +6 ~ +7：80% 成功率
  - +8 ~ +9：60% 成功率
  - +10：40% 成功率
  - 失敗不會降級，但素材和金幣消耗

#### 6.2.3 使魔小屋

- **功能**：
  - **孵化**：使用「使魔核心」+ 素材 + 金幣孵化新使魔
  - **培養**：餵食素材提升使魔經驗
  - **技能教學**：使用卷軸教使魔新技能
  - **進化**：滿足條件後進化使魔
  - **存放**：存放不攜帶的使魔（最多存放 10 隻）
  - **更換**：切換攜帶的使魔（攜帶上限 1 隻）

#### 6.2.4 大圖書館

- **功能**：
  - **技能研究**：消耗「研究點數」+ 素材，合成/研發新魔法
  - **技能突變查詢**：查看已學技能的突變路線與條件
  - **敵人圖鑑**：查看已遭遇敵人的詳細資料（弱點、掉落物）
  - **元素知識**：查看屬性相剋表與詳細倍率
  - **導師筆記**：主線劇情文件收集與閱讀

#### 6.2.5 冒險者酒館

- **功能**：
  - **接取任務**：支線任務板，每次刷新 3~5 個任務
  - **任務類型**：
    - 討伐任務：在指定區域擊敗 X 隻特定敵人
    - 收集任務：收集 X 個特定素材
    - 探索任務：到達指定樓層的特定節點
    - 挑戰任務：在限定回合內擊敗特定敵人
  - **任務獎勵**：金幣、素材、稀有卷軸、飾品圖紙
  - **NPC 對話**：與酒館內的冒險者交談，獲取情報或觸發隱藏任務
- **任務上限**：同時最多接取 **3 個任務**

#### 6.2.6 旅館

- **功能**：
  - **休息**：完全回復 HP 和 MP
  - **存檔**：手動存檔點（遊戲也有自動存檔）
  - **做夢（隨機事件）**：休息時有 20% 機率觸發夢境事件
    - 夢見未知的魔法 -> 獲得隨機技能線索
    - 夢見導師 -> 主線劇情提示
    - 噩夢 -> 醒來後獲得暫時減益但有特殊道具

#### 6.2.7 競技場（50F 中層都市限定）

- **功能**：
  - **挑戰模式**：連續戰鬥不回復，看能撐幾場
  - **Boss 回顧戰**：重新挑戰已擊敗的 Boss（可選難度倍率）
  - **排名**：記錄最佳成績
  - **獎勵**：競技場幣 -> 兌換獨佔飾品/卷軸

#### 6.2.8 附魔師（70F 限定）

- **功能**：為裝備附加元素屬性或特殊效果
  - 法杖附火元素 -> 所有攻擊附帶火屬性
  - 法袍附冰抗 -> 冰系傷害減免 20%
  - 附魔可覆蓋，但之前的附魔會消失

#### 6.2.9 詛咒解除師（60F 限定）

- **功能**：
  - 解除被詛咒的裝備（詛咒裝備屬性強但有負面效果，解除後保留正面效果）
  - 解除探索中獲得的持續性詛咒狀態

## 7. 節點式地牢探索系統

### 7.1 設計概要

地牢探索**完全透過點擊選項**進行，不使用格子地圖或即時移動。每層地牢是一個**節點路線圖**，玩家點擊選擇前進方向與互動方式。

### 7.2 節點地圖生成規則

每個非安全區樓層自動生成一張節點地圖：

```
[入口]
  |
  +--- [戰鬥] ---+--- [事件] ---- [戰鬥] ---+
  |              |                          |
  +--- [寶箱] ---+--- [戰鬥] --- [商人] ----+
                                            |
                                        [中繼點]
                                            |
                           +--- [精英戰] ---+
                           |                |
                           +--- [事件] ---- [祭壇] ---+
                                                      |
                                                   [出口]
```

**生成規則：**
- 地圖分為 **3~4 行**（階段）
- 每行有 **2~3 個節點**
- 相鄰行的節點之間隨機連線（保證每個節點至少有一條路可達）
- 入口和出口各一個，固定在首尾
- 節點類型按權重隨機分配（見 5.3 節）

**路線選擇要素：**
- 玩家可以看到節點類型圖示（但不知道具體內容）
- 部分節點在暗屬性區域顯示為「???」
- 選擇路線 = 選擇風險與報酬的平衡

### 7.3 節點類型詳細設計

#### 7.3.1 戰鬥節點

進入後觸發戰鬥遭遇。

**進入畫面：**
```
+-- 12F 熔岩鍛爐 --------------------------------+
|                                                 |
|  你沿著走廊前進，突然感到一股敵意               |
|  前方的陰影中，有什麼東西在移動...               |
|                                                 |
|  偵測到敵人：                                   |
|  - 火焰精靈 x2  Lv.14                          |
|  - 熔岩蜥蜴 x1  Lv.15                          |
|                                                 |
|  +-------------------+  +-------------------+   |
|  | 迎戰               |  | 嘗試避開           |   |
|  | (進入戰鬥)         |  | (50%成功逃離,      |   |
|  |                    |  |  失敗則被偷襲)     |   |
|  +-------------------+  +-------------------+   |
|  +-------------------+                          |
|  | 使用「隱身卷軸」    |                          |
|  | (100%安全通過,      |                          |
|  |  消耗道具)          |                          |
|  +-------------------+                          |
+-------------------------------------------------+

**戰鬥後：**
- 勝利 -> 獲得經驗值、金幣、隨機掉落物
- 逃跑成功 -> 無獲得，回到節點地圖
- 逃跑失敗 -> 被偷襲（敵人先手 1 回合）後正常戰鬥

#### 7.3.2 事件節點

隨機觸發各類事件，透過選項互動。

**事件類型：**

| 類型 | 頻率 | 說明 |

| 發現類 | 30% | 發現寶物、魔法陣、前人遺跡 |
| 遭遇類 | 20% | 遇到NPC、迷路的冒險者、神秘生物 |
| 陷阱類 | 15% | 魔法陷阱、地板機關、詛咒之門 |
| 謎題類 | 15% | 符文解讀、魔法機關、邏輯謎題 |
| 環境類 | 10% | 特殊地形、元素湧泉、魔力結晶 |
| 劇情類 | 10% | 導師筆記、塔的記憶、主線相關 |

#### 7.3.3 精英戰節點

- **獎勵**：保證掉落一件稀有道具或卷軸

### 7.4 探索中的資源管理

探索過程中，以下資源持續消耗：

- **HP**：戰鬥消耗、陷阱傷害、事件代價
- **MP**：技能消耗、探索選項消耗
- **道具**：消耗品使用
- **技能PP**：每個技能有使用次數限制

**資源補充方式：**
- 中繼點休息（回復 30%）
- 使用藥水（消耗道具欄）
- 部分事件選項
- 回到安全區（完全回復）

**核心張力**：資源有限 -> 是繼續推進還是回安全區？

### 7.5 傳送卷軸機制

- 在探索中隨時可使用「傳送卷軸」返回最近的安全區
- 傳送後，當前樓層的進度**不會保留**（需要重新挑戰該層）
- Boss層不可使用傳送卷軸（進入Boss戰後必須戰勝或死亡）

## 8. 回合制戰鬥系統

### 8.1 戰鬥概要

採用經典的**寶可夢式 1v1 / 1v多 回合制戰鬥**，玩家方（主角+使魔）vs 敵人方（1~3隻敵人）。

### 8.2 回合流程

```
回合開始
  |
  v
[速度排序] -> 決定所有行動者的出手順序
  |
  v
[玩家選擇行動]
  - 使用技能（選擇目標）
  - 使用道具
  - 切換使魔指令（攻擊/防禦/輔助）
  - 逃跑
  |
  v
[使魔自動行動] -> 根據指令AI自動選擇技能
  |
  v
[敵人行動] -> 根據AI選擇技能
  |
  v
[結算階段]
  - 傷害計算
  - 狀態異常判定（施加/解除）
  - 持續傷害/回復結算（燃燒、中毒、再生等）
  - 環境效果結算
  - buff/debuff 回合數 -1
  |
  v
[勝負判定]
  - 所有敵人 HP <= 0 -> 勝利
  - 玩家 HP <= 0 -> 敗北
  - 否則 -> 下一回合
```

### 8.3 速度系統

- 每個角色有「速度」屬性
- 每回合按速度由高到低排序行動
- **速度相同**時隨機決定先後
- **優先度**：某些技能有優先度修正
  - +1 優先度：快速技能（如「快速治癒」「先制攻擊」）
  - +2 優先度：防禦/保護技能
  - -1 優先度：蓄力技能
  - 優先度高的技能無視速度，必定先出手

### 8.4 戰鬥行動詳細

#### 8.4.1 使用技能

- 從已裝備的 **6 個主動技能** 中選擇一個
- 消耗 **MP**（魔力值）和 **PP**（技能使用次數）
- 選擇目標（單體技能需選目標，全體技能自動對全體）
- 技能效果立即計算或延遲（蓄力技能）

**技能選擇介面：**

#### 8.4.2 使用道具

- 從背包中選擇消耗品使用
- 使用道具**不消耗MP**但**佔用一個回合**
- 常用戰鬥道具：
  - 回復藥水（回復HP）
  - 魔力藥水（回復MP）
  - 解毒草（解除中毒）
  - 萬靈藥（解除所有狀態異常）
  - 力量藥劑（攻擊力+30%，3回合）
  - 防禦藥劑（防禦力+30%，3回合）
  - 煙霧彈（逃跑成功率100%）

#### 8.4.3 使魔指令

玩家可以切換使魔的行動模式（切換不消耗回合，但每回合只能切換一次）：

| 指令 | 使魔行為 |
|------|---------|
| **攻擊** | 優先使用攻擊技能 |
| **防禦** | 替主角承受下一次攻擊（一次性） |
| **輔助** | 優先使用回復/增益技能 |
| **待命** | 不行動（保存PP） |

#### 8.4.4 逃跑

- 基礎逃跑成功率：**40%**
- 速度高於敵人：+20%
- 速度低於敵人：-10%
- 使用煙霧彈：100%
- Boss戰：**不可逃跑**
- 逃跑失敗：本回合無行動，敵人正常行動

### 8.5 傷害計算公式

```
基礎傷害 = (攻擊力 x 技能威力 / 100) - (對方防禦力 / 2)
屬性修正 = 基礎傷害 x 屬性倍率（見第9章）
暴擊修正 = 屬性修正 x 暴擊倍率（暴擊時 x1.5，否則 x1.0）
隨機修正 = 暴擊修正 x random(0.85, 1.00)
最終傷害 = max(1, 隨機修正)  // 最少造成1點傷害
```

**攻擊力區分：**
- **魔法攻擊力（M.ATK）**：影響魔法技能傷害
- **物理攻擊力（P.ATK）**：影響物理技能傷害（少數技能）
- **魔法防禦力（M.DEF）**：減免魔法傷害
- **物理防禦力（P.DEF）**：減免物理傷害

### 8.6 命中與迴避

```
命中率 = 技能命中率 x (自身命中修正 / 對方迴避修正)
最終命中 = clamp(命中率, 5%, 100%)  // 最低5%命中，最高100%
```

- 部分技能為「必中」（無視迴避）
- 部分技能為「一擊必殺」（命中率固定30%，命中則直接擊敗）

### 8.7 暴擊系統

```
暴擊率 = 基礎暴擊率(5%) + 裝備加成 + 技能加成 + buff加成
暴擊倍率 = 1.5（基礎）+ 飾品加成
```

- 部分技能有「高暴擊率」特性（暴擊率+25%）
- 飾品可以提升暴擊率或暴擊傷害

### 8.8 狀態異常系統

## 10. 魔法技能系統

### 10.1 技能概要

技能是本遊戲的**核心收集要素**，目標提供 200+ 個技能供玩家收集與構築 build。

**技能格限制：**
- 主動技能格：**6 格**
- 被動技能格：**2 格**
- 技能背包（已學但未裝備）：**無上限**

### 10.2 技能獲取途徑

| 途徑 | 說明 | 取得難度 |
|------|------|---------|
| **魔法卷軸** | 地牢掉落/商店購買，使用後直接學會 | 低~中 |
| **魔法書** | Boss掉落/圖書館合成，可反覆研讀提升熟練度 | 中~高 |
| **被動領悟** | 被敵人技能擊中後 5% 機率領悟 | 低（隨機） |
| **圖書館研究** | 消耗研究點數+素材，合成新魔法 | 中 |
| **任務獎勵** | 完成特定任務獲得 | 固定 |
| **事件獎勵** | 探索中特定事件選項 | 隨機 |
| **使魔共鳴** | 使魔進化時可能傳授技能給主角 | 稀有 |

### 10.3 技能成長系統

每個技能有**熟練度等級**（Lv.1 ~ Lv.MAX）：

```
學會技能 -> Lv.1（基礎版）
  |
  使用技能/研讀魔法書 累積熟練度
  |
  v
Lv.2（威力/效果小幅提升）
  |
  v
Lv.3（威力/效果中幅提升，可能解鎖附加效果）
  |
  v
Lv.MAX（完全精通）
  |
  v
突變（分支進化二選一，不可逆）
```

**熟練度獲取：**
- 戰鬥中使用該技能：+1~3 熟練度（擊殺敵人+1額外）
- 研讀對應魔法書：+10 熟練度
- 圖書館研究加速：+5 熟練度（消耗研究點數）

**升級所需熟練度：**
- Lv.1 -> Lv.2：50 熟練度
- Lv.2 -> Lv.3：120 熟練度
- Lv.3 -> Lv.MAX：250 熟練度
- Lv.MAX -> 突變：500 熟練度 + 特殊素材

### 10.4 技能突變系統

達到 Lv.MAX 後，滿足條件可進行**突變**（技能進化）。每個技能有**兩條突變路線**，選擇後不可逆。

**範例：火球術的突變路線**

```
火球術 Lv.MAX
  |
  +--[突變A] 隕石術
  |   威力: 65 -> 120
  |   範圍: 單體 -> 全體
  |   MP: 12 -> 35
  |   附加: 30%機率燃燒
  |   冷卻: 2回合
  |
  +--[突變B] 爆裂火球
      威力: 65 -> 150
      範圍: 單體
      MP: 12 -> 28
      附加: 必定燃燒 + 防禦下降
      冷卻: 無
```

**突變條件：**
- 基礎條件：熟練度達到 500
- 特殊素材：如「火之結晶 x5」（隕石術）或「爆裂核心 x3」（爆裂火球）
- 部分突變需要額外條件：如「累計用火球術擊敗 100 隻敵人」

- 回復 PP 方式：
  - 回到安全區（全回復）
  - 使用「以太」道具（回復單個技能 PP）
  - 中繼點研讀魔法書（回復該書對應技能 PP）

## 13. 使魔系統

### 13.1 使魔概要

使魔是主角的戰鬥夥伴，在戰鬥中自動行動（由 AI 控制）。玩家可透過指令切換使魔的行為模式。

**基礎規則：**
- 攜帶上限：**1 隻**（進入地牢前選擇）
- 存放上限：**10 隻**（使魔小屋）
- 使魔有獨立的 HP、屬性、技能
- 使魔 HP 歸零 -> 退場（本次探索不可再出場，回安全區自動回復）

### 13.2 使魔獲取

**獲取方式：**
1. 擊敗特定敵人 -> 掉落「使魔核心」
2. 攜帶使魔核心回到安全區「使魔小屋」
3. 消耗素材 + 金幣進行「孵化」
4. 孵化後獲得對應使魔

**使魔核心掉落率：**
- 普通敵人：1%
- 精英敵人：5%
- Boss：15%（首次擊敗 100%）

### 13.3 使魔類型

| 類型 | 特點 | 幾何外觀 | 代表使魔 |
|------|------|---------|---------|
| **攻擊型** | 高攻擊，低防禦 | 紅色尖銳三角形/菱形 | 火焰小鬼、雷光精靈 |
| **防禦型** | 高防禦，可替主角擋傷 | 藍色方形/六角形 | 岩石守衛、冰晶護衛 |
| **輔助型** | 每回合提供回復/增益 | 綠色圓形/橢圓 | 森林妖精、光之精靈 |
| **速攻型** | 極高速度，連擊能力 | 黃色星形/閃電形 | 風之使者、影子刺客 |
| **特殊型** | 獨特能力（偵測/掉落等） | 紫色星形/不規則形 | 虛空觀測者、幸運貓 |

### 13.5 使魔成長

#### 等級提升
- 使魔跟隨主角戰鬥獲得經驗值（主角獲得量的 50%）
- 升級提升基礎屬性
- 每 10 級解鎖一個里程碑獎勵

#### 餵養培養
- 在使魔小屋消耗素材餵養使魔
- 不同素材提升不同屬性：
  - 力量果實 -> M.ATK +2
  - 守護果實 -> M.DEF +2, P.DEF +2
  - 速度果實 -> 速度 +2
  - 生命果實 -> HP +10
  - 元素結晶 -> 對應元素傷害 +3%
- 每隻使魔有餵養上限（防止無限堆疊）：每種屬性最多 +20

#### 技能教學
- 使用魔法卷軸可以教使魔新技能
- 使魔技能格只有 **2 個**，需要覆蓋舊技能
- 使魔只能學習與自身元素相同或無屬性的技能

### 13.6 使魔進化

滿足條件後可在使魔小屋進行進化：

```
進化條件 = 等級要求 + 特殊素材

範例進化鏈：
火焰小鬼 (Lv.1) 
  -> Lv.25 + 火之結晶x10 
  -> 獄炎魔 (進化)
     -> Lv.45 + 熔岩核心x5 + 龍鱗x3
     -> 獄炎魔王 (最終進化)
```

**進化效果：**
- 大幅提升基礎屬性
- 外觀變化（更大/更複雜的幾何形狀）
- 可能習得新技能
- 可能改變/增加元素屬性
- 部分進化路線有分支（二選一）

### 13.7 使魔戰鬥 AI

使魔根據玩家設定的指令自動行動：

**攻擊模式：**
```
1. 優先使用屬性相剋有利的攻擊技能
2. 如果沒有有利屬性，使用威力最高的攻擊技能
3. PP 不足時使用普通攻擊
```

**防禦模式：**
```
1. 本回合替主角承受下一次攻擊（傷害由使魔承受）
2. 如果使魔 HP < 30%，自動切換為待命模式
```

**輔助模式：**
```
1. 主角 HP < 50% 時優先使用回復技能
2. 否則使用增益技能
3. 沒有輔助技能時使用普通攻擊
```

**待命模式：**
```
不行動，節省 PP
```

## 15. Boss設計

### 15.1 Boss設計原則

- 每個Boss有**獨特機制**，需要特定策略應對
- Boss戰**不可逃跑**
- Boss有多階段行為切換
- 首次擊敗Boss有**保底獎勵**（稀有卷軸/飾品/使魔核心）

## 18. 傳送與進度系統

### 18.1 傳送門機制

- 每次擊敗Boss後，該Boss層的傳送門**永久啟動**
- 已啟動的傳送門可以在任意安全區的傳送陣使用
- 傳送門允許直接傳送到對應安全區（跳過中間樓層）

**傳送門列表：**

| 傳送門 | 啟動條件 | 目的地 |
|--------|---------|--------|
| 1F傳送門 | 初始解鎖 | 1F 廢墟前哨站 |
| 10F傳送門 | 擊敗石像守衛 | 10F 安全區 |
| 20F傳送門 | 擊敗熔岩巨蛇 | 20F 安全區 |
| 30F傳送門 | 擊敗冰霜女巫 | 30F 安全區 |
| 40F傳送門 | 擊敗風暴飛龍 | 40F 安全區 |
| 50F傳送門 | 擊敗大地巨像 | 50F 安全區 |
| 60F傳送門 | 擊敗雷神雙子 | 60F 安全區 |
| 70F傳送門 | 擊敗暗影領主 | 70F 安全區 |
| 80F傳送門 | 擊敗聖騎士長 | 80F 安全區 |
| 90F傳送門 | 擊敗虛空吞噬者 | 90F 安全區 |

### 18.2 樓層進度

- 每個樓層的通關狀態獨立記錄
- 通過一層（到達出口）= 解鎖下一層的入口
- Boss層需要擊敗Boss才算通過
- **樓層不會變化**：每次重新挑戰同一層，節點地圖會重新生成，但敵人等級和類型範圍不變

### 18.3 存檔系統

- **自動存檔**：每次進入/離開安全區時自動存檔
- **手動存檔**：在安全區可隨時手動存檔
- **存檔欄位**：3 個存檔欄
- **探索中不可存檔**（Roguelike 特性）
- 探索中退出遊戲 = 進度丟失，回到最近安全區

## 19. 死亡與懲罰機制

### 19.1 死亡觸發條件

- 主角 HP <= 0 時觸發死亡
- 使魔退場不算死亡（只是使魔不可用）

### 19.2 死亡懲罰

採用**中度懲罰**設計，有損失但不會讓玩家絕望：

```
死亡後：
  1. 傳送回最近的安全區（已啟動的傳送門）
  2. HP/MP 全回復
  3. 使魔 HP 全回復
  
  懲罰：
  - 失去當前攜帶金幣的 20%
  - 本次探索獲得的未鑑定道具全部消失
  - 已鑑定的道具保留
  - 技能PP全回復（算是好處）
  - 當前樓層進度重置（需重新挑戰）
```

### 19.3 死亡保護機制

有多種方式減輕或避免死亡懲罰：

| 保護方式 | 效果 | 取得 |
|---------|------|------|
| 守護天使飾品 | HP首次歸零時回復30%（每場1次） | 聖光區Boss |
| 靈魂鏈接飾品 | 致命傷由使魔代替承受 | 隱藏任務 |
| 復活卷軸 | 死亡時自動使用，HP回復50% | 80F+商店/掉落 |
| 保險箱（1F） | 存入的金幣不受死亡影響 | 安全區NPC |
| 祝福狀態 | 死亡時保留所有道具（不失去未鑑定品） | 聖光殿堂祭壇 |

## 20. UI/UX 設計

#### 節點地圖生成演算法

```gdscript
# floor_generator.gd
func generate_floor(floor_number: int) -> NodeMapData:
    var map = NodeMapData.new()
    var rows = randi_range(3, 4)
    var cols_per_row = []
    
    for r in rows:
        cols_per_row.append(randi_range(2, 3))
    
    # 生成節點
    map.add_node(NodeType.ENTRANCE, 0, 0)
    for r in rows:
        for c in cols_per_row[r]:
            var type = _roll_node_type(floor_number)
            map.add_node(type, r + 1, c)
    map.add_node(NodeType.EXIT, rows + 1, 0)
    
    # 保證中繼點
    _ensure_rest_node(map, rows)
    
    # 生成連線（保證可達性）
    for r in range(rows):
        _connect_adjacent_rows(map, r, r + 1)
    
    return map

func _roll_node_type(floor: int) -> NodeType:
    var weights = {
        NodeType.BATTLE: 35,
        NodeType.EVENT: 20,
        NodeType.CHEST: 15,
        NodeType.ELITE: 5 + floor / 10,
        NodeType.MERCHANT: 10,
        NodeType.REST: 10,
        NodeType.ALTAR: 5,
    }
    return _weighted_random(weights)
```

## 附錄C：數值公式與平衡附錄

### C.1 傷害計算公式

#### 基礎傷害公式

```
基礎傷害 = (技能威力 x 攻擊方M.ATK / 防禦方M.DEF) x 屬性倍率 x 隨機修正

其中：
  攻擊方M.ATK = 基礎M.ATK + 裝備加成 + buff加成
  防禦方M.DEF = 基礎M.DEF + 裝備加成 + buff加成
  屬性倍率 = 查表（見 9.3 元素相剋表）
  隨機修正 = 0.85 ~ 1.15（均勻分布）
```

#### 暴擊傷害

```
暴擊判定 = rand() < 暴擊率
暴擊傷害 = 基礎傷害 x 1.5（預設暴擊倍率）

暴擊率 = 基礎暴擊率 + 裝備加成 + buff加成
暴擊率上限 = 80%
```

#### 物理傷害公式

```
物理傷害 = (技能威力 x 攻擊方P.ATK / 防禦方P.DEF) x 隨機修正
（不受屬性倍率影響）
```

#### 命中判定

```
命中率 = 技能命中率 x (攻擊方命中 / (攻擊方命中 + 防禦方迴避))

命中率下限 = 10%（除特殊技能外，總有最低命中率）
必中技能：命中率 = 100%（無視迴避）
```

### C.2 屬性成長公式

#### 主角升級屬性

```
HP(Lv)  = 100 + (Lv - 1) x 12
MP(Lv)  = 60  + (Lv - 1) x 8
M.ATK(Lv) = 15 + (Lv - 1) x 3
M.DEF(Lv) = 10 + (Lv - 1) x 2
P.ATK(Lv) = 8  + (Lv - 1) x 1
P.DEF(Lv) = 8  + (Lv - 1) x 1.5
速度(Lv)  = 15 + (Lv - 1) x 1.5
命中(Lv)  = 100 + (Lv - 1) x 0.5
迴避(Lv)  = 5  + (Lv - 1) x 0.3

範例（Lv.50）：
HP  = 100 + 49 x 12 = 688
MP  = 60  + 49 x 8  = 452
M.ATK = 15 + 49 x 3 = 162
M.DEF = 10 + 49 x 2 = 108
```

#### 經驗值需求

```
所需EXP(Lv) = 50 + (Lv^2) x 5

Lv.1→2:  55
Lv.10→11: 550
Lv.50→51: 12,550
Lv.99→MAX: 49,055
```

#### 使魔升級經驗

```
使魔所需EXP(Lv) = 30 + (Lv^2) x 3
使魔獲得EXP = 主角獲得EXP x 0.5
```

### C.3 裝備強化公式

#### 強化成功率

```
成功率(+N) = max(10%, 100% - N x 10%)

+0 → +1: 100%
+1 → +2: 90%
+2 → +3: 80%
+3 → +4: 70%
+4 → +5: 60%
+5 → +6: 50%
+6 → +7: 40%
+7 → +8: 30%
+8 → +9: 20%
+9 → +10: 10%
```

#### 強化數值加成

```
法杖強化：
  M.ATK加成(+N) = 基礎M.ATK x (N x 0.08)
  即每+1 = 基礎攻擊力的 8%

法袍強化：
  M.DEF加成(+N) = 基礎M.DEF x (N x 0.08)
  P.DEF加成(+N) = 基礎P.DEF x (N x 0.08)
  HP加成(+N)    = 基礎HP加成 x (N x 0.05)
```

#### 強化費用

```
強化費用(+N) = 基礎費用 x (1 + N x 0.5)

普通裝備基礎費用: 100G
稀有裝備基礎費用: 300G
史詩裝備基礎費用: 800G
傳說裝備基礎費用: 2000G
```

### C.4 經濟平衡

#### 金幣收入

```
戰鬥金幣 = 基礎金幣 x (1 + 樓層 / 50) x 難度修正
Boss金幣 = 基礎金幣 x 10

每區域預期金幣收入/層（普通戰鬥平均）：
1F~9F:    15G/戰
11F~19F:  25G/戰

41F~49F:  75G/戰

61F~69F:  130G/戰
71F~79F:  170G/戰
81F~89F:  220G/戰
91F~99F:  300G/戰
```

#### 金幣支出

```
商品價格隨樓層調整：
  商品價格 = 基礎價格 x (1 + 解鎖安全區層數 / 100)

每區域預期支出（回復藥+強化+技能研究）：
  設計目標：收入略大於支出（約 1.2 倍）
  確保玩家不會因正常遊玩而破產
  死亡懲罰（20%金幣）不應造成惡性循環
```

### C.5 掉落率公式

```
最終掉落率 = 基礎掉落率 x (1 + 幸運加成) x 稀有度修正

幸運加成來源：
  - 幸運貓使魔：+20%
  - 掉落率飾品：+10%~+30%
  - 稱號加成：+10%~+20%
  - 祭壇祝福：+15%（臨時）

稀有度修正（對應稀有度的額外乘數）：
  N:   x1.0
  R:   x0.8
  SR:  x0.5
  SSR: x0.3
  UR:  x0.1
```
